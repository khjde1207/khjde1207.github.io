ဘယ်လိုသုံးရမလဲ

ဒီလျှောက်လွှာမှာလုပ်ဆောင်ချက်တွေအများကြီးရှိတယ်, ဒါကြောင့်မင်းကိုဖတ်ဖို့အကြံပေးလိုပါတယ်

1. ဖိုင်များကိုကြည့်ရှုရန်
  - ကျေးဇူးပြု. ကျွန်ုပ်၏စာရွက်စာတမ်းရှိ Add File ကိုနှိပ်ပါ။
  
  - TXT ဖိုင်ကိုနှိပ်ပြီးဖိုင်ကိုကြည့်ရန်အောက်ခြေရှိ Open ခလုတ်ကိုနှိပ်ပါ။


2. ကြည့်ရှုသူကြော်ငြာများ / အောက်ခြေလမ်းညွှန်ခလုတ်များဖုံးကွယ်နည်း
  - မျက်နှာပြင်၏အလယ်ဗဟိုကိုနှစ်ချက်နှိပ်ပါ။
  
  - မျက်နှာပြင်အပြည့်ကိုဖွင့်သောအခါကြော်ငြာများနှင့်ခလုတ်များပျောက်ကွယ်သွားသည်။


3. Mitt function area ရိယာကိုပြောင်းလဲရန်။
  - ၎င်းကိုချိန်ညှိချက်များအောက်တွင်ပြောင်းလဲနိုင်သည် - Touch area ရိယာချိန်ညှိချက်များအောက်တွင်ပြောင်းလဲနိုင်သည်။
  
  - မျက်နှာပြင်အပြည့်, စာမျက်နှာလှုပ်ရှားမှု, တောက်ပမှုညှိနှိုင်းမှုနှင့်အသံအတိုးအကျယ်ထိန်းချုပ်ခြင်းအပါအဝင်လုပ်ဆောင်ချက်များစွာကိုသင်သတ်မှတ်နိုင်သည်။
  
  - Bluetooth ဝေးလံခေါင်ဖျားသောထိန်းချုပ်မှုကိုလည်းထောက်ခံသည်။


4. စာလုံးသို့မဟုတ်စာလုံးအရွယ်အစားကိုပြောင်းလဲရန်
  - ချိန်ညှိချက်များအောက်တွင်စာလုံးသို့မဟုတ်စာလုံးအရွယ်အစားကိုပြောင်းလဲနိုင်သည် - ချိန်ညှိချက်များ - Settings - font settings ကိုသင်ပြောင်းလဲနိုင်သည်။
  
  - အခမဲ့ဖောင့်များ၏ဒါဇင်ပေါင်းများစွာကိုလက်ခံရရှိရန် download ခလုတ်ကိုနှိပ်ခြင်းဖြင့်ဖောင့်ကိုသုံးနိုင်သည်။


5. ပုံစာသားပြောင်းခြင်းကိုမည်သို့အသုံးပြုရမည်နည်း။
  - သင်သည်စာသားကို compressed image မှထုတ်ယူနိုင်သည်။
  
  - ၎င်းကို OCR function ဟုခေါ်သည်။
  
  - ကျွန်ုပ်၏ဂျာနယ် - Add File ကိုနှိပ်ပြီး zip file တစ်ခုကိုထည့်ပါ။
  
  - သင်သည် zip ဖိုင်ကိုရွေးချယ်ပြီး OCR menu ကိုရွေးချယ်ခြင်းဖြင့်ပြောင်းလဲနိုင်သည်။
  
  - ပြောင်းလဲခြင်းအတွင်းစာမျက်နှာကိုစားသုံးသည်။

  - OCR သည် AI နှင့်အလုပ်လုပ်သည်။ ထို့ကြောင့်အမြန်နှုန်းနှင့်တိကျမှုသည်အလွန်ကောင်းသည်။


6. EPUB, PDF ကူးပြောင်းခြင်းနည်းလမ်း:
  - ၎င်းသည်ပုံကူးပြောင်းခြင်းနည်းလမ်းနှင့်အတူတူပင်ဖြစ်သည်။

  - သတိထားရမည့်အရာအနည်းငယ်ရှိသည်။

  - EPUB သို့မဟုတ် PDF ကို "စာဝှက်ထား" နိုင်သည်။

  - ၎င်းသည်အဓိကအားဖြင့်အက္ခရာများကိုဖြိုခွဲရန်နှင့်အချို့သောဖောင့်များတွင်သာမြင်နိုင်အောင်ပြုလုပ်ရန်အသုံးပြုသည်။

  - ဤကိစ္စတွင်ကိုးရီးယားမဟုတ်ဘဲ extraterRestrial ဇာတ်ကောင်ကဲ့သို့ထွက်လာသည်, ထို့ကြောင့်၎င်းကို TTS တွင်မဖတ်နိုင်ပါ။

  - ဤပြ problem နာကိုဖြေရှင်းရန်နည်းလမ်းတစ်ခုအနေဖြင့်ပုံတစ်ပုံကိုဖြည်ချခြင်း၏လုပ်ဆောင်ချက်ကိုသုံးနိုင်သည်။ ထို့နောက် OCR ကို၎င်းကိုထုတ်ယူရန်လှည့်နိုင်သည်။


7. TTS ကိုဘယ်လိုသတ်မှတ်ရမလဲ။
  - TTS-Related settings ကိုချိန်ညှိချက်များတွင်အာရုံစိုက်ထားပါသည်။ TTS settings, TTS filter settings ကို TTS STSE STSE STRESS တွင်ဖော်ပြထားသည်။
  
  - TTS setting များတွင်အသံ, အမြန်နှုန်း, အသံအတိုးအကျယ်, ခလုတ်များပွင့်လင်းမြင်သာမှု, စာကြောင်းများအဆုံးသတ်ခြင်းနှင့်ဝါကျများကိုအချိန်မှန်ဖြင့်ဖတ်သည့်စာကြောင်းများ,
  
  - တစ်ချိန်တည်းတွင်စာကြောင်းများဖတ်သည့်စာကြောင်းများမှာအနိမ့်ဆုံးနှောင့်နှေးမှုဖြင့်ကြားနာရန်တစ်ပြိုင်နက်တည်းဖတ်ရှုခွင့်ပြုရန်ကဆက်နွယ်သည်။

  - TTS filter သည်ဤလျှောက်လွှာ၏အင်အားအကောင်းဆုံးအင်္ဂါရပ်များထဲမှတစ်ခုဖြစ်သည်။
  
  - [TTS filter settings] သည်သင် skip လုပ်ရန်လိုအပ်သည့်အထူးအက္ခရာများ, စကားလုံးများသို့မဟုတ်စာကြောင်းများကိုသတ်မှတ်ရန်ခွင့်ပြုသည်။
  
  - သင် download ခလုတ်ကိုနှိပ်လျှင်, ငါထုတ်လုပ်သည့် download download လုပ်နိုင်သော download လုပ်နိုင်သည်။
  
  - သင်လိုအပ်လျှင်သင်ကိုယ်တိုင်ထည့်နိုင်သည်။

  - [အလိုအလျောက်ပိတ်ခြင်းလုပ်ဆောင်ချက်] TTS ကစားသည့်အခါ Stop ခလုတ်၏ဘယ်ဘက်ခြမ်းရှိ Mook-shaped icon ကိုထိခြင်းဖြင့်အလိုအလျောက်ပိတ်ချိန်ကိုသတ်မှတ်နိုင်သည်။


8. အရန်ကူးခြင်းနှင့်ပြန်လည်ထူထောင်ရေးစွမ်းရည်များအကြောင်းဖော်ပြချက် -
  - Setup - Backup နှင့် Recovery သည်ကိရိယာမျိုးစုံကိုတူညီသောပတ် 0 န်းကျင်တွင်အသုံးပြုရန်ခွင့်ပြုသည့်အင်္ဂါရပ်တစ်ခုဖြစ်သည်။
  
  - Backup Recovery အတွက် drive သို့သင်ဝင်ရောက်နိုင်သည်။
  
  - PC များ, ဖုန်းများနှင့် pads များကဲ့သို့သောကိရိယာများစွာမှကြည့်ရှုသောအခါသင်၏ကြည့်ရှုနေရာကိုတစ်ပြိုင်တည်းချိန်ကိုက်ရန်တည်နေရာ backup ကိုသင်အသုံးပြုနိုင်သည်။
  
  - သင် application ကိုဖွင့ ်. အိမ်သို့ပြန်သွားသည့်အခါအလိုအလျောက်ရွှေ့ပြောင်းရန်အလိုအလျောက်ရွေ့လျားရန်အလိုအလျောက်အလိုအလျောက်ရွေ့လျားရန်အလိုအလျောက်ပြုလုပ်လိုသည့်ကိရိယာကိုရွေးချယ်ပါ။


9. စာသားအသုတ်ပြုပြင်ခြင်းကိုဘယ်လိုသုံးရမလဲ။
  - 1. [set-set-set-show-edit button]
  
  - 2. [ချိန်ညှိချက်များ - Bult Text Modifictify]
  
  - အသုတ်ပြုပြင်မွမ်းမံကိုနည်းလမ်းနှစ်မျိုးဖြင့်သင်အသုံးပြုနိုင်သည်။
  
  - နံပါတ် 1 ဖြစ်လျှင်၎င်းသည် TTS ကိုနားထောင်ရန်နှင့်ဝါကျများအားဝါကျများကိုအခြားစာလုံးများသို့မဟုတ်စကားလုံးများအနှောင့်အယှက်ဖြစ်စေရန်သို့မဟုတ်ပြောင်းလဲရန်သို့မဟုတ်ပြောင်းလဲရန်အသုံးပြုသည်။
  
  - နံပါတ် 2 ဖြစ်လျှင် OCR နှင့်စာသားကို Extract လုပ်နိုင်ပြီးအသုတ်ဆုံးမခြင်းကိုပြုလုပ်ရန်စာရွက်စာတမ်းမှစာသားဖိုင်ကိုရွေးချယ်နိုင်သည်။
  
  - စာသားအစုအပွင့်ဆုံးမခြင်းအတွက်မကြာခဏဖြစ်ပျက်နေသောအသိအမှတ်ပြုအမှားများကို download လုပ်ရန် download ခလုတ်ကိုသုံးနိုင်သည်။
  
  - အမြောက်အများပြင်ဆင်ချက် - ဒေါင်းလုပ်များကသာလျှင်ဘာသာစကားချိန်ညှိချက်များသည်ကိုရီးယားတွင်ရှိလျှင်စာရင်းကိုသာပြလိမ့်မည်။
  
  - ဤအချက်မှာ developer များအခြားဘာသာစကားများမပြောသောကြောင့်ဖြစ်သည်။
  
  - အကယ်. သင်သည်အစီရင်ခံစာ function ကို အသုံးပြု. သတင်းပို့ပါကကျွန်ုပ်တိုင်းပြည်၏ဘာသာစကားကိုထည့်ပါမည်။


10. TTS ကိုနားထောင်သည့်အခါဝါကျတစ်ဝါကျရှိလျှင်၎င်းကိုလုပ်ဆောင်ပါ။
  - မူရင်းပြုပြင်မွမ်းမံခြင်းရှိမရှိအရေးမကြီးပါက၎င်းကို (စုပေါင်းပြုပြင်မွမ်းမံခြင်း) function သို့ပြောင်းပါ။

  - အပြောင်းအလဲပြီးနောက်အရန်ကူးဖိုင်သည်ကျွန်ုပ်၏စာကြည့်တိုက်တွင်ဆက်လက်ရှိနေလိမ့်မည်။

  - အကယ်. သင်သည်မထိဘဲ TTS များမှသာလျှင် TTS မှသာဖယ်ရှားလိုပါက TTS filter ကိုသုံးပါ။



11. စာမျက်နှာနှင့်ဆက်စပ်သောဖော်ပြချက်:
  - အကယ်. သင်သည် TTS ဝါကျစစ်ဆေးခြင်း, အသုတ်ပြင်ဆင်ခြင်း, လိုင်းများကိုပြန်လည်ဖွဲ့စည်းခြင်း, ပြန်လည်ဖွဲ့စည်းခြင်း, ဘာသာပြန်များကိုပြန်လည်ဖွဲ့စည်းခြင်း,

  - OCR, EPUB နှင့် PDF မှစာသားထုတ်ယူသည့်အခါစာမျက်နှာအများစုကိုခံစားရန်အသုံးပြုလိမ့်မည်။

  - စာသားကိုထုတ်ယူသောအခါကျွန်ုပ်တို့သည်တစ်ခုတည်းသောလက်ကိုဆန့်ကျင်ခြင်းသာမကပုံစံများကိုပုံစံများဖြင့်ထုတ်ယူပြီးနောက်ပေါ်ထွက်လာသောပြ problems နာများကိုပြုလုပ်သောပြ problems နာများစွာပြုလုပ်ခြင်းဖြင့်စာကြောင်းများကိုလည်းစုစည်းထားသည်။

  - ဒါဟာအကောင်အထည်ဖော်ရန်အလွန်ခဲယဉ်းသည်, ဒါကြောင့်မင်းရဲ့နားလည်မှုကိုတောင်းခံခဲ့တယ်။


12. ကြော်ငြာဖော်ပြချက် -
  - ငါက developer နဲ့အသုံးပြုသူနှစ်ယောက်စလုံးပါ။

13. နားကြပ်အတွက်ဖော်ပြချက် (သမိုင်းအပြန်အလှန်ဆွေးနွေးခြင်း):

  - စာသားဖိုင်တစ်ခုကိုကျွန်ုပ်၏စာကြည့်တိုက်တွင်ရွေးသောအခါနား View Menu ပေါ်လာသည်။
  
  - အကယ်. သင်သည်ဤမီနူးကိုရွေးချယ်ပြီး (1 မှ 50) ကိုအရင်ရွေးချယ်ပါကယခင်ကကြည့်ရှုသည့်အနေအထားကိုအလိုအလျောက်ကျော်သွားလိမ့်မည်။

  - သင်အချိန်ရှိလျှင်သမိုင်းမှ (1-50) ကိုဖယ်ရှားပါ။

  - သမိုင်းတွင်ဖတ်ထားသောစာအုပ်အရေအတွက်ကိုလည်းရေတွက်သည်။
  
  - အကယ်. တူညီသောစာရွက်စာတမ်းများကျန်ရှိနေပါကတိကျသောသတင်းအချက်အလက်များကိုပြသနိုင်သည်။



14. Bookmark / မှတ်စု / သရုပ်ဆောင်ဖော်ပြချက်:
  - ဤအက်ပလီကေးရှင်းသည်မှတ်စုများနှင့် bookmark / starring ကိုထောက်ပံ့သည်။

  - အကယ်. သင်သည် bookmark settings တွင် bookmark function ကိုဖွင့်ပါကစာသားကို နှိပ်. ကိုင်ထားခြင်းဖြင့်၎င်းကိုထည့်နိုင်သည်။

  - သင်အနည်းဆုံးတစ်ကြိမ်ဖွင့်လှစ်ခဲ့ပါကဖိုင်ကိုရွေးချယ်ရန်နှင့်ထည့်ရန်သင့်အားဖိုင်တစ်ခုရွေးရန်ခွင့်ပြုသည်။

  - အကယ်. သင်သည်ဖိုင်ကိုအနည်းဆုံးတစ်ကြိမ်ဖွင့်ပါက File Selection ၏ထိပ်တွင်ကြယ်တစ်လုံးပေါ်လာသည်။

  - မှတ်တမ်းများနှင့် Horoscopes ဖိုင်ကိုအကဲဖြတ်ရန်မှတ်တမ်းတင်ရန်အသုံးပြုနိုင်သည်။

  - Bookmarks များသည်အရေးကြီးသောအကြောင်းအရာများကိုမှတ်တမ်းတင်ရန်အသုံးပြုနိုင်သည်။
