၎င်းကိုမည်သို့အသုံးပြုရမည် (ဖတ်ရမည်):

ဒီလျှောက်လွှာမှာလုပ်ဆောင်ချက်တွေအများကြီးရှိတယ်, ဒါကြောင့်မင်းကိုဖတ်ဖို့အကြံပေးလိုပါတယ်

1. ဖိုင်များကိုကြည့်ရှုရန်
ကျေးဇူးပြု. ကျွန်ုပ်၏စာရွက်စာတမ်းရှိ Add File ကိုနှိပ်ပါ။
txt ဖိုင်ကိုနှိပ်ပြီးဖိုင်ကိုကြည့်ရန်အောက်ခြေရှိ Open ခလုတ်ကိုနှိပ်ပါ။


2. ကြည့်ရှုသူကြော်ငြာများ / အောက်ခြေလမ်းညွှန်ခလုတ်များဖုံးကွယ်နည်း
မျက်နှာပြင်၏အလယ်ဗဟိုကိုနှစ်ချက်နှိပ်ပါ။
မျက်နှာပြင်အပြည့်ကိုဖွင့်လိုက်တာနဲ့ကြော်ငြာတွေနဲ့ခလုတ်တွေပျောက်သွားတယ်။


3. Mitt function area ရိယာကိုပြောင်းလဲရန်။
၎င်းကိုချိန်ညှိချက်များအောက်တွင်၎င်းကိုပြောင်းလဲနိုင်သည် - Touch area ရိယာချိန်ညှိချက်များအောက်တွင်ပြောင်းလဲနိုင်သည်။
မျက်နှာပြင်အပြည့်, စာမျက်နှာလှုပ်ရှားမှု, တောက်ပမှုညှိနှိုင်းမှုနှင့်အသံအတိုးအကျယ်ထိန်းချုပ်ခြင်းအပါအ 0 င်လုပ်ဆောင်ချက်များစွာကိုသင်သတ်မှတ်နိုင်သည်။
Bluetooth ဝေးလံခေါင်ဖျားသောထိန်းချုပ်မှုကိုလည်းထောက်ပံ့သည်,


4. စာလုံးသို့မဟုတ်စာလုံးအရွယ်အစားကိုပြောင်းလဲရန်
Settings - Settings - Settings - font size အောက်ရှိစာလုံးသို့မဟုတ်စာလုံးအရွယ်အစားကိုပြောင်းလဲနိုင်သည်။
download ခလုတ်ကိုနှိပ်ခြင်းဖြင့် font ကို သုံး. အခမဲ့ဖောင့်များကိုဖတ်ရန် download ခလုတ်ကိုနှိပ်ပါ။


5. ပုံစာသားပြောင်းခြင်းကိုမည်သို့အသုံးပြုရမည်နည်း။
သငျသညျ compressed ပုံမှစာသားထုတ်ယူနိုင်ပါတယ်။ ဒါကို OCR function လို့ခေါ်တယ်။
ကျွန်ုပ်၏ဂျာနယ် - Add File ကိုနှိပ်ပြီး zip file တစ်ခုထည့်ပါ။
သင်သည် zip ဖိုင်ကိုရွေးချယ်ပြီး OCR menu ကိုရွေးချယ်ခြင်းဖြင့်ပြောင်းလဲနိုင်သည်။
ပြောင်းလဲခြင်းအတွင်းစာမျက်နှာကိုစားသုံးသည်။
OCR သည် AI နှင့်အလုပ်လုပ်သည်, ထို့ကြောင့်၎င်း၏အမြန်နှုန်းနှင့်တိကျမှုသည်အတော်အတန်ကောင်းသည်။


6. EPUB, PDF ကူးပြောင်းခြင်းနည်းလမ်း:
၎င်းသည်ပုံကူးပြောင်းခြင်းနည်းလမ်းနှင့်အတူတူပင်ဖြစ်သည်။
သတိထားရမည့်အချက်အနည်းငယ်ရှိသည်။
EPUB သို့မဟုတ် PDF ကို "encrypted" ဖြစ်နိုင်သည်။
၎င်းကိုအဓိကအားဖြင့်စာလုံးများကိုဖြိုခွဲရန်နှင့်အချို့သောဖောင့်များတွင်သာမြင်နိုင်အောင်ပြုလုပ်ရန်အသုံးပြုသည်။
ဤကိစ္စတွင်ကိုးရီးယားမဟုတ်ဘဲ extracesestrial ဇာတ်ကောင်ကဲ့သို့ထွက်လာသည်, ထို့ကြောင့်၎င်းကို TTS တွင်မဖတ်နိုင်ပါ။
ဤပြ problem နာကိုဖြေရှင်းရန်နည်းလမ်းတစ်ခုအနေဖြင့်ပုံများကိုထုတ်ယူခြင်း၏လုပ်ဆောင်မှုကို အသုံးပြု. ၎င်းကို အသုံးပြု. OCR ကိုထုတ်ယူရန်လှည့်နိုင်သည်။


7. TTS ကိုဘယ်လိုသတ်မှတ်ရမလဲ။
TTS-Related settings များကိုချိန်ညှိချက်များ - TTS Settings, TTS စစ်ဆင်ခြင်းများ,
TTS setting များတွင်အသံ, မြန်နှုန်း, အသံအတိုးအကျယ်, ခလုတ်များပွင့်လင်းမြင်သာမှု,
တစ်ချိန်တည်းတွင်ဖတ်သည့်စာကြောင်းများအရေအတွက်မှာအသေးစားနှောင့်နှေးမှုနှင့်အတူကြားနာရန်တစ်ပြိုင်နက်တည်းဖတ်ရှုခွင့်ပြုရန်ကဆက်ပြောသည်။
TTS filter သည်ဤလျှောက်လွှာ၏အင်အားအကောင်းဆုံးအင်္ဂါရပ်များထဲမှတစ်ခုဖြစ်သည်။
[TTS filter settings] သည်သင် skip လုပ်ရန်လိုအပ်သည့်အထူးအက္ခရာများ, စကားလုံးများသို့မဟုတ်ဝါကျများကိုသတ်မှတ်ရန်ခွင့်ပြုသည်။
download ခလုတ်ကိုနှိပ်လျှင်, ငါထုတ်လုပ်သော download download လုပ်နိုင်သော download လုပ်နိုင်သည်။
သင်လိုအပ်လျှင်၎င်းကိုသင်ထည့်နိုင်သည်။
[အလိုအလျောက်ပိတ်ခြင်းလုပ်ဆောင်ချက်] TTS ကစားသည့်အခါ Stop ခလုတ်၏ဘယ်ဘက်ခြမ်းရှိ Mook-shaped icon ကိုထိခြင်းဖြင့်အလိုအလျောက်ပိတ်ချိန်ကိုအလိုအလျောက်ပိတ်ထားနိုင်သည်။


8. အရန်ကူးခြင်းနှင့်ပြန်လည်ထူထောင်ရေးစွမ်းရည်များအကြောင်းဖော်ပြချက် -
Setup - Backup နှင့် Recovery သည်ပစ္စည်းမျိုးစုံကိုတူညီသောပတ် 0 န်းကျင်ရှိအသုံးပြုရန်ခွင့်ပြုသည့်အင်္ဂါရပ်တစ်ခုဖြစ်သည်။
backup ပြန်လည်ထူထောင်ရေးအတွက် drive ကို log in လုပ်နိုင်ပါတယ်။
PC များ, ဖုန်းများနှင့် pads ကဲ့သို့သောကိရိယာများကိုကြည့်ရှုသောအခါသင်၏ကြည့်ရှုနေရာကိုတစ်ပြိုင်တည်းချိန်ကိုက်ရန်နေရာအရံသိမ်းဆည်းခြင်းကိုသင်အသုံးပြုနိုင်သည်။
သင် Autosynchronize လုပ်လိုသည့်ကိရိယာကိုရွေးချယ်ပြီးလျှောက်လွှာကိုဖွင့ ်. အိမ်သို့ပြန်သွားသည့်အခါနောက်ဆုံးနေရာသို့အလိုအလျောက်ရွှေ့ရန် Autosynchronization feature ကိုဖွင့်ပါ။

9. စာသားအသုတ်ပြုပြင်ခြင်းကိုဘယ်လိုသုံးရမလဲ။
1. [ချိန်ညှိချက်များ - ချိန်ညှိချက်များ - Settings - Bulk Edit Button]
2. [ချိန်ညှိချက်များ - စာသားများကိုအတူတကွတည်းဖြတ်ခြင်း]
သငျသညျနည်းလမ်းနှစ်ခုအတွက်အသုတ်ပြုပြင်မွမ်းမံကိုသုံးနိုင်သည်။
နံပါတ် 1 ဖြစ်လျှင် TTS ကိုနားထောင်ရန်နှင့်ဝါကျများကိုအခြားစာလုံးများသို့မဟုတ်စကားလုံးများအနှောင့်အယှက်ဖြစ်ခဲ့လျှင်ဝါကျကိုအစားထိုးရန်သို့မဟုတ်ပြောင်းလဲရန်အသုံးပြုသည်။
နံပါတ် 2 အလိုတွင်သင်သည် OCR နှင့်စာသားကို Extract လုပ်နိုင်ပြီးအသုတ်ဆုံးမပဲ့ပြင်ရန်စာရွက်စာတမ်းမှစာသားဖိုင်ကိုရွေးချယ်နိုင်သည်။
စာသား၏ပြင်ဆင်မှုအတွက်မကြာခဏဖြစ်ပျက်နေသောအမှားအယွင်းများကိုဒေါင်းလုပ်ဆွဲရန်နှင့်အသုံးပြုရန် download ခလုတ်ကိုသုံးနိုင်သည်။
အမြောက်အများပြင်ဆင်ချက် - Downloads သည်ဘာသာစကားချိန်ညှိချက်များသည်ကိုရီးယားတွင်ရှိလျှင်စာရင်းကိုသာပြလိမ့်မည်။
ဘာဖြစ်လို့လဲဆိုတော့ developer တွေကတခြားဘာသာစကားမပြောတတ်လို့ပါ။
အစီရင်ခံစာ function ကိုအသုံးပြုပြီးသင်အစီရင်ခံပါကကျွန်ုပ်သည်တိုင်းပြည်၏ဘာသာစကားကိုထည့်ပါလိမ့်မည်။

10. TTS ကိုနားထောင်သည့်အခါဝါကျတစ်ဝါကျရှိလျှင်၎င်းကိုလုပ်ဆောင်ပါ။
မူရင်းပြုပြင်မွမ်းမံခြင်းရှိမရှိအရေးမကြီးပါက ကျေးဇူးပြု. ၎င်းကို (စုပေါင်းပြုပြင်မွမ်းမံခြင်း) function သို့ပြောင်းပါ။
အပြောင်းအလဲအပြီးတွင်အရန်ကူးဖိုင်သည်ကျွန်ုပ်၏စာကြည့်တိုက်တွင်ဆက်လက်ရှိနေလိမ့်မည်။
အကယ်. သင်သည်မူရင်းကို TTS များမှသာဖယ်ရှားလိုပါက TTS filter ကိုသုံးပါ။

11. စာမျက်နှာနှင့်ဆက်စပ်သောဖော်ပြချက်:
အကယ်. သင်သည် TTS ဝါကျစစ်ဆေးခြင်း, အသုတ်ဆုံးမပဲ့ပြင်ခြင်း, လိုင်းများကိုပြန်လည်ဖွဲ့စည်းခြင်း,
OCR, EPUB နှင့် PDF မှစာသားထုတ်ယူသောအခါစာမျက်နှာအများစုကိုခံစားရန်အသုံးပြုလိမ့်မည်။
စာသားကိုထုတ်ယူသောအခါကျွန်ုပ်တို့သည်တစ်ခုတည်းသောလက်ကိုသာထုတ်ယူရုံသာမကဘဲထုတ်ယူခြင်းသည်ပုံစံများကိုထုတ်ယူပြီးနောက်ပေါ်ထွက်လာသောပြ problems နာများစွာပြုလုပ်ခြင်းဖြင့်ဝါကျများကိုစုစည်းထားသည်။
၎င်းသည်အကောင်အထည်ဖော်ရန်အလွန်ခက်ခဲသောအစိတ်အပိုင်းတစ်ခုဖြစ်ပြီး ကျေးဇူးပြု. ကြိုးစားပါ။


12. ကြော်ငြာဖော်ပြချက် -
ငါက developer နဲ့အသုံးပြုသူနှစ် ဦး စလုံးပါ။
ကျွန်ုပ်တို့သည်၎င်းတို့အားတတ်နိုင်သမျှအသုံးမပြုသည့်အတိုင်းအတာအထိကြော်ငြာများကိုထည့်သွင်းထားသည်။
ကြော်ငြာများတွင်စာမျက်နှာလုံခြုံသောကြော်ငြာများနှင့်ကြော်ငြာဖယ်ရှားရေးကြော်ငြာများပါဝင်သည်။
