Ungayisebenzisa kanjani:

Lolu hlelo lokusebenza lunemisebenzi eminingi, ngakho-ke ngincoma ukuthi uyifunde ngakunye.

1. Ungawahlola kanjani amafayela:
  - Sicela uchofoze Faka ifayela kudokhumenti yami ukuze ungeze i-TXT.
  
  - Thinta ifayela le-TXT bese ucindezela inkinobho evulekile ngezansi ukuze ubuke ifayela.


2. Ungayifihla Kanjani Izikhangiso Zomhleli / Izinkinobho Zokuhambisa Ezingaphansi:
  - Thepha kabili maphakathi nesikrini.
  
  - Njengoba ushintshela esikrinini esigcwele, izikhangiso nezinkinobho ziyanyamalala.


3. Ukushintsha indawo yokusebenza kwenkinobho:
  - Ungashintsha ngaphansi kwezilungiselelo - amasethingi endawo yokuthinta.
  
  - Ungasetha eminye imisebenzi, kufaka phakathi isikrini esigcwele, ukunyakaza kwekhasi, ukulungiswa kokukhanya, nokulawulwa kwevolumu.
  
  - Ukulawulwa okukude kwe-Bluetooth nakho kuyasekelwa, ngakho-ke ungalala phansi bese uyibuka kahle.


4. Ukushintsha usayizi wefonti noma yefonti:
  - Ungashintsha usayizi wefonti noma yefonti ngaphansi kwezilungiselelo - Izilungiselelo - Izilungiselelo zefonti.
  
  - Ungasebenzisa ifonti ngokucindezela inkinobho yokulanda ukuthola inqwaba yamafonti wamahhala.


5. Ungayisebenzisa kanjani ukuguqulwa kombhalo wesithombe:
  - Ungakhipha umbhalo kusuka esithombeni esicindezelwe.
  
  - Lokhu kubizwa ngokuthi umsebenzi we-OCR.
  
  - Ijenali yami - chofoza okuthi Faka ifayela bese wengeza ifayela le-zip.
  
  - Ungaguqula ngokukhetha ifayela le-zip bese ukhetha imenyu ye-OCR.
  
  - Ikhasi liyadliwa ngesikhathi sokuguqulwa.

  - I-OCR isebenza nge-AI, ngakho-ke ijubane nokunemba kukuhle impela.


6. I-EPUB, indlela yokuguqula ye-PDF:
  - Kuyafana nendlela yokuguqulwa kwesithombe.

  - Kunezinto ezimbalwa okufanele wazi.

  - I-EPUB noma i-PDF ingahle ibekwe "ngemfihlo".

  - Isetshenziselwa ikakhulukazi ukuphula izinhlamvu futhi kubenze babonakale kuphela kumafonti athile.

  - Kulokhu, kuphuma njengomlingiswa ongemva kokungemthetho, hhayi isiKorea, ngakho-ke awukwazi ukuyifunda kuma-TTS.

  - I-solver ingakhipha lokhu ngokusebenzisa umsebenzi wokuyikhipha njengesithombe ngendlela, bese ngokuguqula i-OCR.


7. Ukusetha kanjani ama-TTS:
  - Izilungiselelo ezihlobene ne-TTS zigxile kuzilungiselelo - Izilungiselelo ze-TTS, izilungiselelo zokuhlunga ze-TTS.
  
  - Izilungiselelo ze-TTS zifaka izici ezilula ze-TTS, njengezwi, isivinini, ivolumu, ukuvela kwenkinobho, izinhlamvu eziqeda imisho, kanye nenombolo yemisho efundwa ngesikhathi, kanye namadivayisi ahambisana nezici ze-TTS.
  
  - Inani lemisho efundwa ngesikhathi umsebenzi owengeziwe ongeziwe ukuze uvumele imigqa eminingi ukuthi ifundwe ngaso leso sikhathi ukuze izwakale ngokubambezeleka okuncane.

  - Isihlungi se-TTS ngesinye sezici ezinamandla kakhulu zalolu hlelo lokusebenza.
  
  - [Amasethingi e-TTS Finger] ikuvumela ukuthi usethe inani lezinhlamvu ezikhethekile, amagama noma imisho okudingeka yeqe.
  
  - Uma ucindezela inkinobho yokulanda, ungalanda izinqwaba zezihlungi engizikhiqizile.
  
  - Ungangeza futhi ngokwakho uma uyidinga.

  - [Umsebenzi wokuvala othomathikhi] ungasetha isikhathi sokuvala esizenzakalelayo ngokuthinta isithonjana esenziwe ngenyanga ngakwesokunxele inkinobho yokumisa lapho udlala ama-TTS.


8. Ukuchazwa kwe-Backup kanye namakhono wokutakula:
  - Setup - Isipele nokululama kuyisici esivumela amadivayisi amaningi ukuthi asebenzise uhlelo lokusebenza endaweni efanayo.
  
  - Ungangena ngemvume kwidrayivu yokubuyiselwa kwesipele.
  
  - Uma kubukwa kusuka kumadivayisi amaningi, njengama-PC, amafoni, namaphakethe, ungasebenzisa isipele sendawo ukuvumelanisa indawo yakho yokubuka.
  
  - Khetha idivaysi ofuna ukuyisebenzisa i-autosynchronize bese uvula isici se-autosynchchronization ukuze uhambise ngokuzenzakalelayo endaweni yokugcina lapho usebenzisa uhlelo lokusebenza noma ubuye.


9. Ungayisebenzisa kanjani ukuguqulwa kombhalo we-batch:
  - 1. [Isethi-Setha-Setha-Setha-Khombisa inkinobho yokuhlela]
  
  - 2. [Izilungiselelo - Ukuguqulwa kombhalo ngobuningi]
  
  - Ungasebenzisa ukuguqulwa kwe-batch ngezindlela ezimbili.
  
  - Uma kwenzeka inombolo 1, isetshenziselwa ukulalela ama-TTS nokusula noma ukuguqula umusho komunye umlingiswa uma eminye imisho noma amagama ayacasula.
  
  - Uma kwenzeka inombolo 2, ungakhipha umbhalo nge-OCR bese ukhetha ifayela lombhalo kusuka kudokhumenti ukwenza ukulungiswa kwe-batch.
  
  - Ungasebenzisa futhi inkinobho yokulanda ukulanda futhi usebenzise amaphutha wokuqashelwa kaningi ngokulungiswa kwe-batch kombhalo.
  
  - Ukulungiswa kwe-Bulk - Ukulanda kuzokhombisa kuphela uhlu uma izilungiselelo zolimi ziseKorea.
  
  - Lokhu kungenxa yokuthi abathuthukisi abakhulumi ezinye izilimi.
  
  - Uma ubika usebenzisa umsebenzi wombiko, ngizongeza ulimi lwezwe.


10. Uma kunomusho wokuhlaselwa lapho ulalele ama-TTS, usebenze:
  - Uma okwangempela kungashintshwa, sicela usishintshe ukuguqulwa kwe-batch.

  - Ngemuva koshintsho, ifayela lesipele lizosala umtapo wezincwadi wami, ngakho-ke uma ulishintsha ngephutha, ungaphinde uthole.

  - Uma ufuna ukususa okwangempela kuphela kusuka kuma-TTS ngaphandle kokuthinta, sebenzisa isihlungi se-TTS.



11. Ikhasi elihlobene Ikhasi elihlobene:
  - Uma usebenzisa imisebenzi ekhethekile efana nokuhlolwa kwemisho ye-TTS, ukulungiswa kwemisho ye-batch, ukuhlelwa kabusha kwemigqa, nokuhumusha, amakhasi azodliwa, kepha njengoba kungama-10 kuye kwangama-50, ngeke uzizwe kudliwe.

  - Amakhasi amaningi kungenzeka asetshenziselwe ukuzizwa lapho ekhipha umbhalo ovela ku-OCR, ePub kanye ne-PDF.

  - Lapho sikhipha umbhalo, asigcini nje ngokukhipha i-single-ngesandla, kepha futhi sihlele imisho ngokwenza izinkinga eziningi ezivela ngemuva kokukhishwa kwamaphethini.

  - Bekunzima ngempela ukusebenzisa, ngakho-ke ngicela ukuqonda kwakho.


12. Incazelo Yokukhangisa:
  - Ngingunjiniyela nobabili nomsebenzisi.

  - Sengeze izikhangiso kuze kufike lapho zingaphazamisi ukusetshenziswa ngangokunokwenzeka.

  - Izikhangiso zifaka nezikhangiso ezivikeleke ekhasini nezikhangiso zokususa izikhangiso.

  - Ikhasi lingabamba amakhasi afinyelela ku-60,000.

  - Lapho ususa izikhangiso, ungasebenzisa ikhasi ukususa izikhangiso isikhathi esithile. (Ngaphandle kwesikhathi sokulala, kucishe kube usuku.)

  - Kulabo amakhasi agcinwe kwangaphambili, ungasusa ukukhangiswa okubukwayo ngokudla amakhasi angama-2000.

  - Uma usebenzisa amakhasi angama-8000, zonke izikhangiso zikhutshaziwe ngaphandle kwazo izikhangiso ezi-30 zesibili zokutholwa kwekhasi nemikhangiso yendabuko phakathi kwezilungiselelo.

  - Izikhangiso zekhasi ezingama-8000 zengezwe zalabo abakhathazwa yizikhangiso ezimfushane ezivela ngesikhathi senani elikhulu lokuguqulwa.


13. Isici Sencazelo Yendlebe (Umlando Ukusebenzelana Nokusebenzelana):
  - Ixhasa umsebenzi wokubuka kwezindlebe.
  
  - Isibonelo, ake sithi sinombhalo onesihloko esithi (1 kuya ku-50), futhi sinombhalo omusha obizwa nge (1 kuye ku-100).
  
  - Kulokhu, sekuvele kudingekile ukweqa isikhundla esikhulu.
  
  - Uma ukhetha ifayela lombhalo kumtapo wezincwadi wami, imenyu yokubuka izindlebe iyavela.
  
  - Uma ukhetha le menyu bese ukhetha (1 kuye ku-50) emlandweni, uzokweqa isikhundla esibukwe ngaphambilini.

  - Uma usunaso isikhathi emuva kwesikhathi, sicela ususe (1-50) emlandweni.

  - Emlandweni, inani eliphelele lezincwadi ezifundwayo libalwa futhi.
  
  - Uma idokhumenti efanayo ihlala, imininingwane engalungile ingakhonjiswa.

  - Uma uvula umsebenzi we-bookmark kuzilungiselelo zebhandiki, ungayengeza ngokucindezela nokubamba umbhalo.

  - Isici samanothi sikuvumela ukuthi ukhethe futhi ufake ifayili uma usuyivulile okungenani kanye.

  - Uma uvula ifayela okungenani kanye, inkanyezi ivela phezulu kokukhethwa kwefayela.

  - Amanothi nezinkanyezi ze-Horoscopes zingasetshenziselwa ukuqopha ukuhlolwa kwefayela uqobo.

  - Amabhukhimakhi angasetshenziselwa ukuqopha okuqukethwe okubalulekile.
