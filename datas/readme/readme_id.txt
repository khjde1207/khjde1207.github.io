Cara menggunakannya (harus dibaca):

Aplikasi ini memiliki banyak fungsi, jadi saya sarankan Anda membacanya satu per satu.

1. Cara Melihat File:
Silakan klik Tambah File di dokumen saya untuk menambahkan txt.
Sentuh file txt dan tekan tombol buka di bagian bawah untuk melihat file.


2. Cara Menyembunyikan Iklan Penampil / Tombol Navigasi Bawah:
Ketuk Double Pusat Layar.
Saat Anda beralih ke layar penuh, iklan dan tombol menghilang.


3. Untuk mengubah Area Fungsi Tombol Mitt:
Anda dapat mengubahnya di bawah Pengaturan - Pengaturan Area Sentuh.
Anda dapat mengatur sejumlah fungsi, termasuk layar penuh, gerakan halaman, penyesuaian kecerahan, dan kontrol volume.
Remote control Bluetooth juga didukung, sehingga Anda dapat berbaring dan menontonnya dengan nyaman.


4. Untuk mengubah ukuran font atau font:
Anda dapat mengubah ukuran font atau font di bawah Pengaturan - Pengaturan - Pengaturan Font.
Anda dapat menggunakan font dengan menekan tombol unduh untuk menerima lusinan font gratis.


5. Cara Menggunakan Konversi Teks Gambar:
Anda dapat mengekstrak teks dari gambar terkompresi. Ini disebut fungsi OCR.
Jurnal Saya - Klik Tambah File dan Tambahkan File Zip.
Anda dapat mengonversi dengan memilih file zip dan kemudian memilih menu OCR.
Halaman ini dikonsumsi selama konversi.
OCR bekerja dengan AI, jadi kecepatan dan akurasinya cukup bagus.


6. EPUB, Metode Konversi PDF:
Itu sama dengan metode konversi gambar.
Ada beberapa hal yang harus diperhatikan.
Epub atau PDF dapat "dienkripsi".
Ini terutama digunakan untuk memecahkan huruf dan membuatnya hanya terlihat di font tertentu.
Dalam hal ini, keluar seperti karakter luar angkasa, bukan bahasa Korea, jadi Anda tidak dapat membacanya di TTS.
Sebagai cara untuk menyelesaikan masalah ini, fungsi mengekstraksi gambar dapat digunakan, dan kemudian OCR dapat diputar untuk diekstrak.


7. Cara Mengatur TTS:
Pengaturan terkait TTS terkonsentrasi dalam pengaturan - pengaturan TTS, pengaturan filter TTS.
Pengaturan TTS mencakup fitur TTS sederhana, seperti suara, kecepatan, volume, transparansi tombol, karakter akhir kalimat, dan jumlah kalimat yang dibaca sekaligus, serta perangkat yang melengkapi fitur TTS.
Jumlah kalimat yang dibaca pada satu waktu adalah fungsi yang ditambahkan untuk memungkinkan beberapa baris dibaca sekaligus didengar dengan penundaan minimal.
Filter TTS adalah salah satu fitur paling kuat dari aplikasi ini.
[Pengaturan Filter TTS] memungkinkan Anda untuk mengatur sejumlah karakter, kata atau kalimat khusus yang perlu Anda lewati.
Jika Anda menekan tombol unduh, Anda dapat mengunduh lusinan filter yang saya hasilkan.
Anda juga dapat menambahkannya sendiri jika Anda membutuhkannya.
[Fungsi shutdown otomatis] dapat mengatur waktu shutdown otomatis dengan menyentuh ikon berbentuk bulan di sisi kiri tombol berhenti saat memutar TTS.


8. Deskripsi kemampuan cadangan dan pemulihan:
Pengaturan - Cadangan dan Pemulihan adalah fitur yang memungkinkan beberapa perangkat untuk menggunakan aplikasi di lingkungan yang sama.
Anda dapat masuk ke drive untuk pemulihan cadangan.
Jika dilihat dari beberapa perangkat, seperti PC, ponsel, dan bantalan, Anda dapat menggunakan cadangan lokasi untuk menyinkronkan lokasi tampilan Anda.
Pilih perangkat yang ingin Anda utosinkronisasi dan nyalakan fitur autosinkronisasi untuk secara otomatis pindah ke lokasi terakhir saat Anda menjalankan aplikasi atau pulang dan kembali.

9. Cara Menggunakan Modifikasi Batch Teks:
1. [Pengaturan - Pengaturan - Tombol Edit Massal]
2. [Pengaturan - Edit teks secara kolektif]
Anda dapat menggunakan modifikasi batch dengan dua cara.
Dalam hal nomor 1, digunakan untuk mendengarkan TTS dan menghapus atau mengubah kalimat ke karakter lain jika beberapa kalimat atau kata menjengkelkan.
Dalam hal nomor 2, Anda dapat mengekstrak teks dengan OCR dan memilih file teks dari dokumen untuk melakukan koreksi batch.
Anda juga dapat menggunakan tombol unduh untuk mengunduh dan menggunakan kesalahan pengenalan yang sering terjadi untuk koreksi teks.
Koreksi Massal - Unduhan hanya akan menampilkan daftar jika pengaturan bahasa dalam bahasa Korea.
Ini karena pengembang tidak berbicara bahasa lain.
Jika Anda melaporkan menggunakan fungsi laporan, saya akan menambahkan bahasa negara.

10. Jika ada kalimat yang menggelegar saat mendengarkan TTS, kerjakan:
Jika tidak masalah jika aslinya dimodifikasi, harap ubah ke fungsi [modifikasi kolektif].
Setelah perubahan, file cadangan akan tetap di perpustakaan saya, jadi jika Anda secara tidak sengaja memodifikasinya, Anda dapat memulihkannya lagi.
Jika Anda ingin menghapus yang asli hanya dari TTS tanpa menyentuhnya, gunakan filter TTS.

11. Halaman Keterangan Terkait:
Jika Anda menggunakan fungsi khusus seperti inspeksi kalimat TTS, koreksi batch, reorganisasi garis, dan terjemahan, halaman akan dikonsumsi, tetapi karena sekitar 10 hingga 50, Anda tidak akan merasa dikonsumsi.
Sebagian besar halaman cenderung digunakan untuk merasakan saat mengekstraksi teks dari OCR, EPUB, dan PDF.
Saat mengekstraksi teks, kami tidak hanya mengekstrak sendirian, tetapi juga mengatur kalimat dengan membuat lusinan masalah yang muncul setelah ekstraksi menjadi pola.
Ini adalah bagian yang sangat sulit untuk diimplementasikan, jadi silakan coba.


12. Deskripsi Iklan:
Saya baik pengembang dan pengguna.
Kami telah menambahkan iklan sejauh mereka tidak mengganggu penggunaan sebanyak mungkin.
Iklan termasuk iklan halaman-aman dan iklan penghapusan iklan.
Halaman dapat menampung hingga 60.000 halaman.
Saat menghapus iklan, Anda dapat menggunakan halaman untuk menghapus iklan untuk jangka waktu tertentu. (Tidak termasuk waktu tidur, ini sekitar sehari.)
Bagi mereka yang memiliki halaman pra-penumpukan, Anda dapat menghapus iklan sisi pemirsa dengan mengonsumsi 2000 halaman.
Saat Anda mengonsumsi 8000 halaman, semua iklan dinonaktifkan kecuali untuk iklan 30 detik untuk akuisisi halaman dan iklan asli di tengah pengaturan.
Iklan 8000 halaman telah ditambahkan untuk mereka yang terganggu oleh iklan pendek yang muncul selama sejumlah besar konversi.


13. Deskripsi Fitur untuk Telinga (Sejarah Interworking):
Ini mendukung fungsi tampilan telinga.
Misalnya, katakanlah kami memiliki dokumen berjudul (1 hingga 50), dan kami memiliki dokumen baru yang disebut (1 hingga 100).
Dalam hal ini, sudah perlu untuk melewatkan posisi utama.
Saat Anda memilih file teks di perpustakaan saya, menu tampilan telinga muncul.
Jika Anda memilih menu ini dan kemudian pilih (1 hingga 50) dalam riwayat, itu akan secara otomatis melewatkan posisi yang dilihat sebelumnya.
Ketika Anda punya waktu nanti, silakan hapus (1-50) dari sejarah.
Dalam sejarah, jumlah total buku yang dibaca juga dihitung.
Jika dokumen yang sama tetap, informasi yang tidak akurat dapat ditampilkan.


14. Bookmark/note/Starring Description:
Aplikasi ini mendukung catatan dan bookmark/dibintangi.
Jika Anda menyalakan fungsi bookmark dalam pengaturan bookmark, Anda dapat menambahkannya dengan menekan dan menahan teks.
Fitur Notes memungkinkan Anda untuk memilih dan memasukkan file jika Anda telah membukanya setidaknya sekali.
Jika Anda membuka file setidaknya sekali, bintang muncul di bagian atas pemilihan file.
Catatan dan horoskop dapat digunakan untuk merekam evaluasi file itu sendiri.
Bookmark dapat digunakan untuk merekam konten penting.

15. Penjelasan Fungsi Administrasi Anjing:
Ini adalah metode yang dapat digunakan jika kalimat belum selesai tetapi entri turun.
Biasanya, ini banyak terjadi pada dokumen yang bekerja dengan konversi OCR.
Fitur ini bukan hanya kombinasi dari Enter.
Ini adalah fungsi yang diimplementasikan dengan menganalisis dokumen dengan banyak kesalahan pembukaan dan mengatur pola.
Saya mencoba memperkenalkan AI untuk mengaturnya, tetapi itu tidak mudah.
Ada banyak penelitian tentang pemrosesan bahasa alami hari ini, jadi saya akan menambahkannya jika saya bisa memprosesnya dengan AI nanti.

16. Fungsi Terjemahan Deskripsi:
Fungsi terjemahan terjemahan / browser web lokal didukung.
Terjemahan lokal adalah fungsi yang diterjemahkan menjadi AI yang dikembangkan oleh Google. Kata -kata sederhana dapat diterjemahkan dengan baik, tetapi tampaknya ada banyak kesalahan penerjemahan dalam terjemahan kalimat.
Terjemahan Browser Web adalah fungsi yang menampilkan kalimat yang akan diterjemahkan dalam browser.
Jika Anda menyalakan fungsi terjemahan dalam pengaturan - pengaturan terjemahan dan tekan dan tahan kalimat, fungsi terjemahan akan berfungsi.
