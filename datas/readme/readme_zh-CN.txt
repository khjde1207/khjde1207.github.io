如何使用（必须读）：

该应用程序具有很多功能，因此我建议阅读它。

1.如何查看文件：
请单击我的库中的添加文件以添加TXT。
您可以通过触摸TXT文件并按下底部的打开按钮来查看文件。


2.如何隐藏查看器广告/底部导航按钮：
双击屏幕的中心。
当它切换到全屏时，广告和按钮就会消失。


3.如何更改按钮和功能区域：
您可以在[设置 - 触摸区域设置]中更改它。
您可以设置各种功能，例如全屏，页面移动，亮度控制和音量控制。
还支持蓝牙遥控器，因此您可以将其放在架子上并在躺下时舒适地查看。


4.如何更改字体或文本大小：
您可以更改[设置 - 设置 - 字体设置]中的字体或字体大小。
您可以通过单击“下载”按钮来使用数十个免费字体。


5.如何将图像用于文本转换功能：
您可以从压缩图像中提取文本。这称为OCR函数。
我的库 - 单击添加文件，然后添加zip文件。
您可以通过选择ZIP文件然后选择OCR菜单来转换。
在转换过程中消耗页面。
OCR是通过AI完成的，因此其速度和准确性非常好。


6.如何将EPUB转换为PDF：
它与图像转换方法相同。
有几件事要牢记。
对于epub或pdf，它可以“加密”。
最常见的方法是使字母仅在特定字体中乱和可见。
在这种情况下，它不能使用TTS阅读，因为它看起来像外星人角色，而不是韩语。
要解决此问题，您可以使用图像提取功能，然后使用OCR提取它。


7.如何设置TTS：
与TTS相关的设置集中在[设置-TTS设置]和[TTS过滤器设置]中。
TTS设置不仅包含简单的TTS功能​​，例如语音，速度，音量，按钮透明度，句子结尾字符和句子的数量，还添加了补充TTS函数的设备。
立即读取的句子数量是添加的函数，以允许您一次读取多条线并以最小的延迟收听。
TTS过滤器是此应用程序中最强大的功能之一。
在[TTS过滤器设置]中，您可以设置应跳过的各种特殊字符，单词或句子。
如果您单击“下载”按钮，则可以下载我创建的数十个过滤器。
您也可以在必要时添加自己的。
[自动关闭功能]允许您通过在播放TTS时将月球形图标触摸到停止按钮的左侧来设置自动关闭时间。


8.备份和恢复功能描述：
[设置 - 备份和恢复]是创建的函数，旨在允许您在多个设备上的同一环境中使用该应用程序。
您可以登录驱动器并还原备份。
在多个设备（例如PC，电话或PAD）上查看时，您可以使用位置备份来同步查看位置。
如果您选择一个设备自动同步并打开[自动同步]功能，则在运行应用程序或离开并返回主页时，它将自动移动到最后一个查看的位置。

9.如何使用批处理编辑文本：
1. [设置 - 设置 - 显示批处理编辑按钮]
2. [设置 - 批量编辑文本]
散装编辑可以通过两种方式使用。
在案例1中，如果某些句子或单词在听TTS时很烦人，则用于删除句子或将其更改为其他字符。
在情况2中，您可以使用OCR提取文本，然后从我的库中选择文本文件以执行批处理编辑。
您也可以使用“下载”按钮批量编辑文本以下载经常发生的识别错误。
[批次编辑 - 下载]仅在语言设置为韩文时才显示列表。
这是因为开发人员不知道如何说其他语言。
如果您使用报告功能进行报告，我们将添加您所在国家 /地区的语言。

10.解决方案如果有句子在听TTS时很烦人：
如果您不介意修改原始内容，请使用[批处理编辑]功能更改它。
更改后，备份文件将保留在我的库中，因此，如果您错误地进行更改，则可以还原它们。
如果仅要删除TTS而不触摸原件，请使用TTS过滤器。

11.与点相关的解释：
使用特殊功能（例如TTS句子检查，批处理编辑，线路排列和翻译）时会消耗点，但是由于它们大约是10至50，因此您不会感到消费。
从OCR，EPUB和PDF提取文本时，大多数要点可能会使用的程度。
提取文本时，我们不仅提取文本，而且还会从提取后出现的数十个问题中创建模式，并继续进行句子组织。
该部分非常困难，因此请尝试至少使用一次。


12.广告相关的解释：
我既是开发人员又是用户。
已经添加了广告，以至于它们不会尽可能多地干扰使用。
广告有两种类型：确保广告和删除广告的积分。
您最多可以持有60,000点。
删除广告时，您可以使用点在一定时间段内删除广告。 （不包括睡眠时间，大约一天。）
对于那些提前积累积分的人，您可以通过消耗2000点来删除观众端广告。
当您消耗8000点时，所有广告都将被禁用，除了30秒的广告以确保点和设置中间的本机广告。
为那些发现在执行大型转换任务时弹出的简短广告很烦人的人添加了8000点广告。


13.延续描述（历史链接）功能：
支持连续的查看功能。
例如，假设有一个名为A（1〜50）的文件，但是您现在收到一个名为A（1〜100）的新文件。
在这种情况下，您已经需要跳过主要位置。
当您在我的库中选择文本文件时，出现“继续菜单”。
选择此菜单后，如果您在历史记录中选择A（1〜50），它将自动跳过先前查看的位置。
稍后时间，请从历史记录中删除（1〜50）。
在历史上，读取的书总数也被计算在内。
如果仍然存在相同的文档，则可能会显示不准确的信息。


14.书签/备忘录/星级评级说明：
该应用支持备忘录和书签/星级评级功能。
如果您在书签设置中打开书签功能，则可以长时间按文本来添加它。
备忘录函数允许您至少打开并查看文件，从而选择并输入文件。
如果您甚至一次打开文件，则选择文件时，顶部有一颗星星。
注释和评分可用于记录文件本身的评估。
书签可用于记录重要信息。



14. TTS句子检查器描述：
当我浏览了许多文档时，我开始看到写得很好的文章和不那么井井有条的文章之间的模式。
我发现了一种总是在令人失望的文章中出现的模式。
这是我通过组织这些模式并为每种模式赋予权重来创建的功能。
首先，此功能被命名为[Minesweeper]。
但是，此功能是基于过于主观的判断而创建的，因此已将其更改为TTS句子检查器。
此功能仅支持韩语。
因为我不太了解其他国家的语言，所以我无法弄清楚写作是好是坏。
如果其他国家 /地区有这样的模式，请通过电子邮件进行报告。
一旦我们可以支持多种民族语言，我们将把功能移至设置。
设置中仍然剩下一个空间。


15.线断路功能的描述：
线断路定理是一种方法，当句子未完成但按下Enter键时可以使用。
这种现象通常发生在通过OCR转换处理的文档中。
此功能不仅组合Enter。
该功能是通过分析具有许多Newline错误并组织模式的文档来实现的。
我试图使用AI介绍组织，但这并不容易。
如今，正在对自然语言处理进行大量研究，因此，如果我们将来可以与AI进行处理，我们将尝试添加它。

16.翻译功能描述：
支持本地翻译/Web浏览器翻译功能。
本地翻译是一种使用Google开发的AI转换的函数。简单的单词翻译得很好，但是句子翻译中似乎有很多错误译文。
Web浏览器翻译是一个显示要在浏览器中翻译的句子的函数。
如果您在[设置 - 翻译设置]中打开翻译功能，然后按住句子，则翻译功能将起作用。
