如何使用它：

该应用程序具有很多功能，因此我建议您一一阅读它。

1.如何查看文件：
   - 请单击我的文档中的添加文件以添加TXT。
  
   - 触摸TXT文件，然后按底部的打开按钮以查看文件。


2.如何隐藏查看器广告 /底部导航按钮：
   - 双击屏幕的中心。
  
   - 当您切换到全屏时，广告和按钮消失了。


3.更改按钮MITT功能区域：
   - 您可以在设置下更改它 - 触摸区域设置。
  
   - 您可以设置许多功能，包括全屏，页面移动，亮度调整和音量控制。
  
   - 还支持蓝牙遥控器，因此您可以躺下并舒适地观看。


4.更改字体或字体大小：
   - 您可以在设置 - 设置 - 字体设置下更改字体或字体大小。
  
   - 您可以通过按下载按钮接收数十个免费字体来使用字体。


5.如何使用图像文本转换：
   - 您可以从压缩图像中提取文本。
  
   - 这称为OCR函数。
  
   - 我的日记 - 单击添加文件，然后添加zip文件。
  
   - 您可以通过选择ZIP文件然后选择OCR菜单来转换。
  
   - 该页面在转换过程中被消耗。

   -  OCR与AI一起使用，因此速度和准确性相当好。


6. EPUB，PDF转换方法：
   - 与图像转换方法相同。

   - 有几件事要注意。

  -EPUB或PDF可能被“加密”。

   - 它主要用于打破字母，并使它们仅在某些字体中可见。

   - 在这种情况下，它像外星人的角色一样，而不是韩语，因此您无法在TTS中阅读它。

   - 求解器可以通过以某种方式将其提取为图像的功能，然后通过转动OCR提取。


7.如何设置TTS：
   - 与TTS相关的设置集中在设置中-TTS设置，TTS过滤器设置。
  
   -  TTS设置包括简单的TTS功能​​，例如语音，速度，音量，按钮透明度，字符结束句子以及一次读取的句子数量以及补充TTS功能​​的设备。
  
   - 一次读取的句子数量是添加的函数，以允许以最小的延迟来一次读取多行。

   -  TTS过滤器是此应用程序中最强大的功能之一。
  
   -  [TTS过滤器设置]允许您设置需要跳过的许多特殊字符，单词或句子。
  
   - 如果按下下载按钮，则可以下载我生产的数十个过滤器。
  
   - 如果需要，也可以自己添加。

   -  [自动关闭功能]可以通过在播放TTS时触摸停止按钮左侧的月球形图标来设置自动关闭时间。


8.备份和恢复功能的描述：
   - 设置 - 备份和恢复是一项功能，允许多个设备在同一环境中使用该应用程序。
  
   - 您可以登录到驱动器以进行备份恢复。
  
   - 当从多个设备（例如PC，电话和垫子）查看时，您可以使用位置备份来同步查看位置。
  
   - 选择要自动同步的设备，然后打开AutoSynchronization功能，在运行应用程序或回家并返回时自动移至最后一个位置。


9.如何使用文本批处理修改：
  -1。[设置设定显示“编辑”按钮]
  
  -2。[设置 - 批量修改]
  
   - 您可以通过两种方式使用批处理修改。
  
   - 在数字1的情况下，如果某些句子或单词令人讨厌，它用于聆听TTS，将句子删除或更改为另一个字符。
  
   - 如果是数字2，则可以使用OCR提取文本，然后从文档中选择文本文件以执行批次校正。
  
   - 您还可以使用“下载”按钮下载并使用经常发生的识别错误进行批处理文本校正。
  
   - 批量更正 - 下载仅在语言设置在韩文中时才显示列表。
  
   - 这是因为开发人员不会说其他语言。
  
   - 如果您使用报告功能报告，我将添加该国的语言。


10.如果听TTS时有刺耳的句子，请解决：
   - 如果可以修改原件，请将其更改为批处理修改。

   - 更改后，备份文件将保留在我的库中，因此，如果您不小心修改它，则可以再次恢复它。

   - 如果您只想从TTS中删除原件而不触摸原件，请使用TTS过滤器。



11.页面相关描述：
   - 如果您使用特殊功能，例如TTS句子检查，批处理校正，行重组和翻译，则将消耗页面，但由于它大约是10到50，因此您不会觉得它消耗了。

   - 从OCR，EPUB和PDF提取文本时，大多数页面都可能用于感觉。

   - 提取文本时，我们不仅单手提取，而且还通过在提取成模式后出现的数十个问题来组织句子。

   - 真的很难实施，所以我要求您理解。


12.广告描述：
   - 我既是开发人员又是用户。

   - 我们添加了广告，以至于它们不会尽可能多地干扰使用。

   - 广告包括页面安全广告和广告删除广告。

   - 页面最多可容纳60,000页。

   - 删除广告时，您可以使用该页面在一定时间段内删除广告。 （不包括睡眠时间，大约一天。）

   - 对于那些预先堆叠页面的人，您可以通过消费2000页来删除观众端广告。

   - 当您使用8000页时，所有广告将被禁用，除了30秒的广告供页面获取和设置中间的本机广告。

   - 为那些被大量转换过程中弹出的简短广告困扰的人添加了8000页广告。


13.耳朵的功能描述（历史互动）：
   - 它支持耳视图功能。
  
   - 例如，假设我们有一个名为A（1至50）的文件，我们有一个名为A（1至100）的新文档。
  
   - 在这种情况下，已经有必要跳过主位置。
  
   - 当您在我的库中选择文本文件时，会出现“耳视”菜单。
  
   - 如果选择此菜单，然后在历史记录中选择一个（1至50），则它将自动跳过先前查看的位置。

   - 稍后时间，请从历史记录中删除（1-50）。

   - 在历史上，还计算了读书的总数。
  
   - 如果仍然存在相同的文档，则可能会显示不准确的信息。



14.书签/注/主演说明：
   - 该应用程序支持注释和书签/主演。

   - 如果您在书签设置中打开书签函数，则可以通过按和持有文本来添加它。

   - 注释功能使您至少打开一次文件，可以选择并输入文件。

   - 如果您至少打开文件，则会在文件选择的顶部出现一颗恒星。

   - 笔记和星座可用于记录文件本身的评估。

   - 书签可用于记录重要内容。



14. TTS句子审查员描述：
   - 当我查看了很多文档时，我开始看到一种写得很好且未写过的文本的模式。

   - 我发现了一种总是在不幸的文本中出现的模式。
  
   - 我组织了这种模式，并通过加权每种模式来生产它。

   - 我们首先将其命名为[我的搜索]。
  
   - 但是，此功能是通过过于主观的判断制成的，因此我将其更改为TTS句子检查器。

   - 此功能仅支持韩语。

   - 我不知道其他国家的语言，所以我无法组织我的模式，无论它是好是坏。

   - 如果其他国家 /地区有这样的模式，请通过电子邮件将其发送给我们。
  
   - 一旦您可以支持多种民族语言，我们将把功能移至设置。

   - 设置中仍然剩下一个位置。



15.狗管理功能的解释：
   - 如果句子未完成，但条目已关闭，这是可以使用的方法。

   - 通常，与OCR转换一起使用的文档发生了很多事情。

   - 此功能不仅是Enter的组合。
  
   - 这是通过分析具有许多开放错误并组织模式的文档来实现的函数。

   - 我试图介绍AI来组织它，但这并不容易。
  
   - 如今，关于自然语言处理有很多研究，因此，如果以后可以使用AI进行处理，我将添加它。


16.翻译功能描述：
   - 支持本地翻译 / Web浏览器翻译功能。
  
   - 本地翻译是一种转化为Google开发的AI的函数。简单的单词可以很好地翻译，但是句子翻译中似乎有很多错误翻译。

   -  Web浏览器翻译是一个显示要在浏览器中翻译的句子的函数。

   - 如果您在设置中打开翻译功能 - 翻译设置并按住句子，则翻译功能将起作用。
