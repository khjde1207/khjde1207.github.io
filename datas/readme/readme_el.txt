Πώς να χρησιμοποιήσετε (πρέπει να διαβάσετε):

Αυτή η εφαρμογή έχει πολλά χαρακτηριστικά, γι 'αυτό συνιστώ να διαβάσετε μέσω αυτής.

1. Πώς να προβάλετε αρχεία:
Κάντε κλικ στην επιλογή Προσθήκη αρχείου στη βιβλιοθήκη μου για να προσθέσετε TXT.
Μπορείτε να δείτε το αρχείο αγγίζοντας το αρχείο TXT και πατώντας το κουμπί Ανοίξτε στο κάτω μέρος.


2. Πώς να αποκρύψετε διαφημίσεις προβολής/κουμπιά πλοήγησης στο κάτω μέρος:
Διπλασιάστε το κέντρο της οθόνης.
Όταν μεταβαίνει σε πλήρη οθόνη, εξαφανίζονται οι διαφημίσεις και τα κουμπιά.


3. Πώς να αλλάξετε το κουμπί και την περιοχή λειτουργίας:
Μπορείτε να το αλλάξετε σε [Ρυθμίσεις - Ρυθμίσεις περιοχής αφής].
Μπορείτε να ορίσετε διάφορες λειτουργίες όπως πλήρης οθόνη, κίνηση σελίδας, έλεγχος φωτεινότητας και έλεγχος όγκου.
Υποστηρίζεται επίσης ένα τηλεχειριστήριο Bluetooth, ώστε να μπορείτε να το τοποθετήσετε σε μια στάση και να το δείτε άνετα ενώ ξαπλώνετε.


4. Πώς να αλλάξετε τη γραμματοσειρά ή το μέγεθος κειμένου:
Μπορείτε να αλλάξετε τη γραμματοσειρά ή το μέγεθος της γραμματοσειράς στις [Ρυθμίσεις - Ρυθμίσεις - Ρυθμίσεις γραμματοσειράς].
Μπορείτε να χρησιμοποιήσετε δεκάδες δωρεάν γραμματοσειρές κάνοντας κλικ στο κουμπί λήψης.


5. Πώς να χρησιμοποιήσετε την εικόνα για τη λειτουργία μετατροπής κειμένου:
Μπορείτε να εξαγάγετε κείμενο από συμπιεσμένες εικόνες. Αυτό ονομάζεται λειτουργία OCR.
Η βιβλιοθήκη μου - Κάντε κλικ στην επιλογή Προσθήκη αρχείου και προσθέστε το αρχείο zip.
Μπορείτε να μετατρέψετε επιλέγοντας το αρχείο zip και, στη συνέχεια, επιλέγοντας το μενού OCR.
Οι σελίδες καταναλώνονται κατά τη μετατροπή.
Το OCR γίνεται με το AI, οπότε η ταχύτητα και η ακρίβειά του είναι αρκετά καλή.


6. Πώς να μετατρέψετε το EPUB σε PDF:
Είναι το ίδιο με τη μέθοδο μετατροπής εικόνας.
Υπάρχουν μερικά πράγματα που πρέπει να έχετε κατά νου.
Στην περίπτωση του Epub ή του PDF, μπορεί να είναι "κρυπτογραφημένο".
Η πιο συνηθισμένη μέθοδος είναι να κάνουν τα γράμματα αλλοιωμένα και ορατά μόνο σε συγκεκριμένες γραμματοσειρές.
Σε αυτή την περίπτωση, δεν μπορεί να διαβαστεί χρησιμοποιώντας το TTS επειδή μοιάζει με έναν αλλοδαπό χαρακτήρα και όχι κορεατικά.
Για να λύσετε αυτό το πρόβλημα, μπορείτε να χρησιμοποιήσετε τη λειτουργία εξαγωγής εικόνας και στη συνέχεια να χρησιμοποιήσετε το OCR για να το εξαγάγετε.


7. Πώς να ρυθμίσετε το TTS:
Οι ρυθμίσεις που σχετίζονται με το TTS συγκεντρώνονται στις [Ρυθμίσεις - Ρυθμίσεις TTS] και [Ρυθμίσεις φίλτρου TTS].
Οι ρυθμίσεις TTS περιλαμβάνουν όχι μόνο απλές λειτουργίες TTS όπως φωνή, ταχύτητα, ένταση, διαφάνεια κουμπιών, χαρακτήρας που τελειώνει η πρόταση και ο αριθμός των προτάσεων που διαβάζονται ταυτόχρονα, αλλά και συσκευές που συμπληρώνουν τη λειτουργία TTS.
Ο αριθμός των προτάσεων που διαβάζονται αμέσως είναι μια συνάρτηση που προστίθεται για να σας επιτρέψει να διαβάσετε πολλές γραμμές ταυτόχρονα και να ακούσετε με ελάχιστη καθυστέρηση.
Το φίλτρο TTS είναι ένα από τα πιο ισχυρά χαρακτηριστικά αυτής της εφαρμογής.
Στις [Ρυθμίσεις φίλτρου TTS], μπορείτε να ορίσετε διάφορους ειδικούς χαρακτήρες, λέξεις ή προτάσεις που πρέπει να παραλειφθούν.
Εάν κάνετε κλικ στο κουμπί λήψης, μπορείτε να κατεβάσετε δεκάδες φίλτρα που δημιούργησα.
Μπορείτε επίσης να προσθέσετε το δικό σας εάν είναι απαραίτητο.
[Η αυτόματη λειτουργία τερματισμού λειτουργίας] σας επιτρέπει να ρυθμίσετε τον αυτόματο χρόνο διακοπής λειτουργίας αγγίζοντας το εικονίδιο σχήματος φεγγαριού στα αριστερά του κουμπιού STOP όταν παίζετε TTS.


8. Περιγραφή λειτουργίας αντιγράφων ασφαλείας και ανάκτησης:
[Ρυθμίσεις - Backup and Recovery] είναι μια λειτουργία που δημιουργήθηκε για να σας επιτρέψει να χρησιμοποιήσετε την εφαρμογή στο ίδιο περιβάλλον σε πολλές συσκευές.
Μπορείτε να συνδεθείτε στη μονάδα δίσκου και να επαναφέρετε τα αντίγραφα ασφαλείας.
Κατά την προβολή σε πολλές συσκευές όπως PC, τηλέφωνο ή PAD, μπορείτε να συγχρονίσετε τη θέση προβολής χρησιμοποιώντας το αντίγραφο ασφαλείας τοποθεσίας.
Εάν επιλέξετε μια συσκευή για να συγχρονίσετε αυτόματα και να ενεργοποιήσετε τη λειτουργία [Auto Sync], θα μετακινηθεί αυτόματα στην τελευταία τοποθεσία που προβάλλεται όταν εκτελείτε την εφαρμογή ή θα φύγετε και θα επιστρέψετε στην αρχική σελίδα.

9. Πώς να χρησιμοποιήσετε το κείμενο επεξεργασίας παρτίδας:
1. [Ρυθμίσεις - Ρυθμίσεις - Εμφάνιση κουμπιού επεξεργασίας παρτίδας]
2. [Ρυθμίσεις - Κείμενο επεξεργασίας χύδην]
Η μαζική επεξεργασία μπορεί να χρησιμοποιηθεί με δύο τρόπους.
Στην περίπτωση 1, εάν κάποιες προτάσεις ή λέξεις είναι ενοχλητικές ενώ ακούτε TTS, χρησιμοποιείται για να διαγράψει τις προτάσεις ή να τις αλλάξει σε άλλους χαρακτήρες.
Στην περίπτωση 2, μπορείτε να εξαγάγετε κείμενο χρησιμοποιώντας το OCR και στη συνέχεια να επιλέξετε αρχεία κειμένου από τη βιβλιοθήκη μου για να εκτελέσετε επεξεργασία παρτίδας.
Μπορείτε επίσης να χρησιμοποιήσετε το κουμπί λήψης για να επεξεργαστείτε το κείμενο επεξεργασίας για να κατεβάσετε συχνά σφάλματα αναγνώρισης.
[Batch Edit - Download] θα εμφανίσει μόνο τη λίστα εάν η ρύθμιση γλώσσας είναι κορεατική.
Αυτό οφείλεται στο γεγονός ότι οι προγραμματιστές δεν ξέρουν πώς να μιλούν άλλες γλώσσες.
Εάν το αναφέρετε χρησιμοποιώντας τη λειτουργία αναφοράς, θα προσθέσουμε τη γλώσσα της χώρας σας.

10. Λύση Εάν υπάρχει μια πρόταση που είναι ενοχλητική όταν ακούτε TTS:
Εάν δεν σας πειράζει η τροποποίηση του αρχικού, χρησιμοποιήστε τη λειτουργία [Επεξεργασία παρτίδας] για να την αλλάξετε.
Μετά από αλλαγές, ένα αρχείο αντιγράφων ασφαλείας θα παραμείνει στη βιβλιοθήκη μου, οπότε αν κάνετε αλλαγές κατά λάθος, μπορείτε να τις επαναφέρετε.
Εάν θέλετε να αφαιρέσετε μόνο τα TTs χωρίς να αγγίξετε το πρωτότυπο, χρησιμοποιήστε το φίλτρο TTS.

11. Επεξήγηση σχετικών με το σημείο:
Τα σημεία καταναλώνονται κατά τη χρήση ειδικών λειτουργιών, όπως ο έλεγχος των προτάσεων TTS, η επεξεργασία παρτίδας, η διάταξη γραμμής και η μετάφραση, αλλά επειδή είναι περίπου 10 έως 50, δεν θα αισθανθείτε την κατανάλωση.
Τα περισσότερα από τα σημεία είναι πιθανό να χρησιμοποιηθούν σε κάποιο βαθμό που μπορεί να γίνει αισθητό κατά την εξαγωγή κειμένου από OCR, EPUB και PDF.
Κατά την εξαγωγή κειμένου, δεν εξάγουμε μόνο το κείμενο, αλλά δημιουργούμε επίσης μοτίβα από δεκάδες προβλήματα που προκύπτουν μετά την εξαγωγή και προχωρούν στην οργάνωση των προτάσεων.
Αυτό το μέρος εφαρμόστηκε με μεγάλη δυσκολία, οπότε δοκιμάστε να το χρησιμοποιήσετε τουλάχιστον μία φορά.


12. Διαφήμιση Σχετική Επεξήγηση:
Είμαι και ο προγραμματιστής και ένας χρήστης.
Οι διαφημίσεις έχουν προστεθεί στο βαθμό που δεν παρεμβαίνουν στη χρήση όσο το δυνατόν περισσότερο.
Υπάρχουν δύο τύποι διαφημίσεων: σημεία που εξασφαλίζουν διαφημίσεις και διαφημίσεις απομάκρυνσης διαφήμισης.
Μπορείτε να κρατήσετε έως και 60.000 πόντους.
Κατά την κατάργηση των διαφημίσεων, μπορείτε να χρησιμοποιήσετε σημεία για να καταργήσετε διαφημίσεις για ορισμένο χρονικό διάστημα. (Εξαιρουμένης του χρόνου ύπνου, είναι περίπου μια μέρα.)
Για όσους συσσωρεύουν σημεία εκ των προτέρων, μπορείτε να καταργήσετε τις διαφημίσεις από την πλευρά του προβολέα καταναλώνοντας 2000 βαθμούς.
Όταν καταναλώνετε 8000 πόντους, όλες οι διαφημίσεις θα απενεργοποιηθούν εκτός από τη διαφήμιση των 30 δευτερολέπτων για την εξασφάλιση σημείων και τη εγγενή διαφήμιση στη μέση των ρυθμίσεων.
Η διαφήμιση των 8000 σημείων προστέθηκε για όσους το βρίσκουν ενοχλητικό για να δουν σύντομες διαφημίσεις που εμφανίζονται κατά την εκτέλεση μεγάλων εργασιών μετατροπής.


13. Περιγραφή της λειτουργίας συνέχισης (σύνδεσμος ιστορικού):
Υποστηρίζει τη λειτουργία συνεχούς προβολής.
Για παράδειγμα, ας πούμε ότι υπάρχει ένα έγγραφο με τίτλο A (1 ~ 50), αλλά τώρα λαμβάνετε ένα νέο έγγραφο με τίτλο A (1 ~ 100).
Σε αυτή την περίπτωση, πρέπει ήδη να παραλείψετε την κύρια τοποθεσία.
Όταν επιλέξετε ένα αρχείο κειμένου στη βιβλιοθήκη μου, εμφανίζεται το μενού Continve.
Αφού επιλέξετε αυτό το μενού, εάν επιλέξετε A (1 ~ 50) στο ιστορικό, θα παραλείψει αυτόματα την τοποθεσία που προέκυψε προηγουμένως.
Όταν έχετε χρόνο αργότερα, αφαιρέστε ένα (1 ~ 50) από την ιστορία.
Στην ιστορία, υπολογίζεται επίσης ο συνολικός αριθμός των βιβλίων.
Εάν παραμένει το ίδιο έγγραφο, μπορούν να εμφανιστούν ανακριβείς πληροφορίες.


14. Περιγραφή σελιδοδείκτη/σημείωμα/αστέρι:
Η εφαρμογή υποστηρίζει τις λειτουργίες σημείωμα και σελιδοδείκτη/αστέρι.
Εάν ενεργοποιήσετε τη λειτουργία σελιδοδεικτών στις ρυθμίσεις του σελιδοδείκτη, μπορείτε να την προσθέσετε μακρά πατώντας το κείμενο.
Η συνάρτηση σημείων σάς επιτρέπει να επιλέξετε και να εισαγάγετε ένα αρχείο εάν έχετε ανοίξει και δείτε το αρχείο τουλάχιστον μία φορά.
Εάν ανοίξετε ένα αρχείο ακόμη και μία φορά, υπάρχει ένα αστέρι στην κορυφή όταν επιλέγετε το αρχείο.
Σημειώσεις και αξιολογήσεις μπορούν να χρησιμοποιηθούν για την καταγραφή των αξιολογήσεων του ίδιου του αρχείου.
Οι σελιδοδείκτες μπορούν να χρησιμοποιηθούν για την καταγραφή σημαντικών πληροφοριών.



14. Περιγραφή ελέγχου πρότασης TTS:
Καθώς κοίταξα πολλά έγγραφα, άρχισα να βλέπω μοτίβα ανάμεσα σε καλά γραπτά και όχι τόσο καλά γραφικά άρθρα.
Ανακάλυψα ένα μοτίβο που εμφανίζεται πάντα σε απογοητευτικά άρθρα.
Αυτή είναι μια συνάρτηση που δημιούργησα με την οργάνωση αυτών των μοτίβων και δίνοντας βάρος σε κάθε μοτίβο.
Αρχικά, αυτή η λειτουργία ονομάστηκε [Minesweeper].
Ωστόσο, αυτή η λειτουργία δημιουργήθηκε με βάση την υπερβολικά υποκειμενική κρίση, οπότε άλλαξε σε έλεγχο προτάσεων TTS.
Αυτό το χαρακτηριστικό υποστηρίζει μόνο την Κορεατική.
Επειδή δεν γνωρίζω καλά τις γλώσσες άλλων χωρών, δεν μπορούσα να καταλάβω αν η γραφή ήταν καλή ή κακή.
Εάν υπάρχει ένα μοτίβο όπως αυτό σε άλλες χώρες, παρακαλούμε να το αναφέρετε μέσω ηλεκτρονικού ταχυδρομείου.
Μόλις μπορέσουμε να υποστηρίξουμε πολλές εθνικές γλώσσες, θα μεταφέρουμε το χαρακτηριστικό στις ρυθμίσεις.
Υπάρχει ακόμα ένας χώρος που απομένει στις ρυθμίσεις.


15. Περιγραφή της λειτουργίας διάλειμμα γραμμής:
Το θεώρημα διάλειμμα γραμμής είναι μια μέθοδος που μπορεί να χρησιμοποιηθεί όταν η πρόταση δεν έχει ολοκληρωθεί, αλλά το πλήκτρο Enter πιέζεται.
Το φαινόμενο αυτό συμβαίνει συνήθως σε έγγραφα που υποβάλλονται σε επεξεργασία μέσω μετατροπής OCR.
Αυτή η λειτουργία δεν συνδυάζει μόνο το ENTER.
Αυτή η λειτουργία εφαρμόστηκε με την ανάλυση εγγράφων με πολυάριθμα σφάλματα Newline και την οργάνωση των μοτίβων.
Προσπάθησα να εισαγάγω τη οργάνωση χρησιμοποιώντας AI, αλλά δεν ήταν εύκολο.
Αυτές τις μέρες, πραγματοποιούνται πολλές έρευνες για την επεξεργασία φυσικής γλώσσας, οπότε αν μπορούμε να το επεξεργαστούμε με το AI στο μέλλον, θα προσπαθήσουμε να το προσθέσουμε.

16. Λειτουργία μετάφρασης Περιγραφή:
Υποστηρίζει λειτουργίες μετάφρασης τοπικής μετάφρασης/προγράμματος περιήγησης ιστού.
Η τοπική μετάφραση είναι μια συνάρτηση που μεταφράζεται χρησιμοποιώντας το AI που αναπτύχθηκε από την Google. Οι απλές λέξεις μεταφράζονται καλά, αλλά φαίνεται ότι υπάρχουν αρκετές λανθασμένες μεταφράσεις στη μετάφραση των προτάσεων.
Η μετάφραση του προγράμματος περιήγησης στο Web είναι μια συνάρτηση που εμφανίζει την πρόταση που θα μεταφραστεί στο πρόγραμμα περιήγησης.
Εάν ενεργοποιήσετε τη λειτουργία μετάφρασης στις [Ρυθμίσεις - Ρυθμίσεις μετάφρασης] και πατήστε και κρατήστε την πρόταση, η λειτουργία μετάφρασης θα λειτουργήσει.

17. Άλλα θέματα:
Κατά το άνοιγμα ενός εγγράφου, το EARAPP θα αλλάξει αυτόματα το πλήκτρο ENTER που επαναλαμβάνεται τρεις ή περισσότερες φορές σε δύο φορές.
Οι υπερβολικές εισόδους είναι άβολα να ακούσουν και να βλέπουν μέσω του TTS.
Μερικοί από εσάς μπορεί να το έχετε ήδη παρατηρήσει κατά τη χρήση της εφαρμογής μου για μεγάλο χρονικό διάστημα και να πείτε, "Χμμ; Τώρα που το σκέφτομαι, δεν έχω δει ποτέ να πατήσετε Enter πολύ; " Μερικοί άνθρωποι μπορεί να το σκέφτονται.
Έχουμε εφαρμόσει όσο το δυνατόν περισσότερα χαρακτηριστικά στον θεατή κειμένου.

Έγραψα δεκαεπτά τρόπους για να το χρησιμοποιήσω, αλλά δεν νομίζω ότι έχω γράψει ακόμη και τα μισά από αυτά.
Έχουν παραλειφθεί απλές λειτουργίες όπως το χρώμα του φόντου, η εικόνα φόντου, το χρώμα κειμένου και η μέθοδος στροφής.
Ελπίζω ότι αυτό το άρθρο θα είναι χρήσιμο για πολλούς ανθρώπους.
Αν κοιτάξετε [Ρυθμίσεις - Πληροφορίες και Βοήθεια], υπάρχει ένας σύνδεσμος προς την αίθουσα συνομιλιών του Telegram, οπότε αν έχετε οποιεσδήποτε ερωτήσεις, μπορείτε να έρθετε και να ρωτήσετε.
