사용방법 : 

이 어플은 기능이 상당히 많으니 한번씩 읽어 보시는걸 추천 드립니다. 

1. 파일 보는 방법 :   
  - 내서재 에서 파일 추가를 눌러 txt 추가해 주세요. 
  
  - txt 파일을 터치하고 맨 하단 열기 버튼을 눌러 파일을 볼 수 있습니다. 


2. 뷰어 광고 / 하단 네비게이션 버튼 숨기는 방법 : 
  - 화면 중앙을 더블탭 하세요. 
  
  - 전체 화면으로 전환되면서 광고 및 버튼들이 사라집니다. 


3. 버튼 밎 기능 영역을 변경 하는 방법 :
  - [설정 - 터치영역 설정] 에서 변경 할 수 있습니다.
  
  - 전체화면, 페이지이동, 밝기 조절, 음량 제어 등 여러 기능을 설정 할 수 있습니다.  
  
  - 블루투스 리모컨도 지원 되니 거치대에 거치해놓고 누워서 편히 볼 수 있습니다. 


4. 폰트나 글자 크기를 변경 하는 방법 :
  - [설정 - 설정- 글꼴 설정] 에서 폰트나 글자 크기를 변경 할 수 있습니다. 
  
  - 폰트는 다운로드 버튼을 눌러 수십개의 무료 폰트를 받아 사용 할 수 있습니다. 


5. 이미지 텍스트 변환 기능 사용 방법 :   
  - 압축된 이미지 에서 텍스트를 추출 할 수 있습니다. 
  
  - 이를 OCR 기능이라 합니다. 
  
  - 내서재 - 파일 추가 누른뒤 zip 파일을 추가 해주세요. 
  
  - zip 파일을 선택한뒤 OCR 메뉴 를 선택하면 변환을 할 수 있습니다. 
  
  - 변환시 페이지가 소모 됩니다. 

  - OCR 은 AI 로 작업 되어 속도와 정확도가 상당히 괜찮습니다. 


6. epub, pdf 변환 방법 : 
  - 이미지 변환 방법과 똑같습니다. 

  - 몇가지 주의하셔야 할 점이 있습니다. 

  - epub 나 pdf 의 경우 "암호화" 되어 있을 수 도 있습니다. 

  - 글자를 깨지게 만들고 특정 폰트에서만 보이도록 하는 방법이 주로 사용 됩니다. 

  - 이경우 한글이 아니라 외계 문자 처럼 나오기 때문에 TTS 로 읽지 못하게 됩니다. 

  - 이를 해결사는 방법으로 이미지로 추출 하는 기능을 사용한뒤, OCR 을 돌려 추출 할 수 있습다.  


7. TTS 설정 방법 : 
  - TTS 관련 설정은 [설정 - TTS 설정], [TTS 필터 설정] 에 집중 되어 있습니다. 
  
  - TTS 설정에는 음성, 속도, 볼륨, 버튼 투명도, 문장이 끝나는 문자, 한번에 읽는 문장의수 등 단순 TTS 기능뿐만 아니라 TTS 기능을 보완하는 장치들도 추가 되어 있습니다. 
  
  - 한번에 읽는 문장의 수는 여러 행을 한번에 읽게 하여 최소한의 딜레이로 듣기위해 추가된 기능입니다. 

  - TTS 필터는 이 어플을 막강한 기능중 하나 입니다. 
  
  - [TTS 필터 설정] 에서는 여러가지 특수문자, 건너뛰어야 하는 단어나 문장들을 설정 할 수 있습니다. 
  
  - 다운로드 버튼을 누르면 제가 제작한 수십개의 필터를 다운받을수 있습니다. 
  
  - 필요할 경우 직접 추가 할 수 도있습니다. 

  - [자동 종료 기능] 은 TTS 재생시 중지 버튼 좌측에 달모양 아이콘을 터치하면 자동 종료 시간을 설정 할 수 있습니다. 


8. 백업 및 복구 기능 설명 : 
  - [설정 - 백업 및 복구] 는 여러 장치에서 동일한 환경으로 어플을 사용할 수 있도록 만들어진 기능 입니다. 
  
  - 드라이브에 로그인 하여 백업 복구를 할 수 있습니다. 
  
  - PC 와 폰, 패드 등 여러 장치에서 보실 경우 위치백업 을 이용해 보는 위치를 동기화 할 수 있습니다. 
  
  - 자동 동기화할 장치를 선택해서 [자동 동기화] 기능을 켜시면 어플실행하거나 홈으로 나갔다 복귀할경우 마지막으로 본 위치로 자동 이동 됩니다.  


9. 텍스트 일괄 수정 사용 방법 : 
  - 1. [설정 - 설정 - 일괄 편집 버튼 보이기]
  
  - 2. [설정 - 텍스트 일괄수정]
  
  - 일괄수정은 두가지 방식으로 사용 할 수 있습니다. 
  
  - 1번의 경우 TTS 로 듣다가 일부 문장이나 단어가 거슬릴 경우 해당 문장을 지우거나 다른 문자로 변경 하는 용도로 사용 됩니다. 
  
  - 2번의 경우 OCR 로 텍스트를 추출한뒤 내서재에서 텍스트파일을 선택하여 일괄수정을 수행 할 수 있습니다. 
  
  - 텍스트 일괄수정에도 다운로드 버튼을 이용하여 자주 발생하는 인식오류를 다운받아 사용 할 수 있습니다. 
  
  - [일괄수정 - 다운로드]는 언어 설정이 한국어 일경우에만 리스트가 보이게 됩니다. 
  
  - 이는 개발자가 다른언어를 할줄 모르기 때문입니다. 
  
  - 제보 기능을 이용하여 제보 해 주시면 해당 국가 언어도 추가 하겠습니다. 


10. TTS 로 들을때 거슬리는 문장이 있을 경우 해결 방법 : 
  - 원본이 수정되도 된다면 일괄 수정 으로 변경 해 주세요. 

  - 변경이후에는 내서재에 백업파일이 남게 되니 실수로 수정 한경우에는 다시 복구를 할 수 있습니다. 

  - 원본은 건들지 않고 TTS에서만 제거 하고자 할 경우 TTS 필터를 사용해 주세요. 



11. 페이지 관련 설명 : 
  - TTS 문장 검사, 일괄수정, 개행 정리, 번역 등 특수기능을 사용할경우 페이지가 소모 되지만, 10~50 정도 이다보니 소모되는걸 체감하지 못할듯 합니다. 

  - 대부분 페이지 는 OCR, epub , pdf 에서 텍스트를 추출 할경우 체감될 정도로 사용 될듯 합니다. 

  - 텍스트 추출시 단수히 추출만 하는것이 아닌, 추출후 나오는 수십가지 문제들을 패턴으로 만들어서 문장 정리 작업까지 진행 합니다. 

  - 정말 힘들게 구현한 부분이니 많은 양해 부탁드립니다. 


12. 광고 관련 설명 : 
  - 저는 개발자 이면서도 사용자 입니다. 

  - 최대한 사용에 방해 받지 않는 정도로만 광고를 추가 하였습니다. 

  - 광고는 페이지 확보광고, 광고 제거 광고 가 있습니다. 

  - 페이지는 최대 6만 페이지를 보유 할 수 있습니다. 

  - 광고제거시 페이지를 사용하여 일정시간 광고를 제거 할 수 있습니다. (수면시간을 제외하면 대략 하루정도의 시간입니다.)

  - 미리 페이지를 쌓아 놓으시는 분을 위해 2000 페이지를 소모 하시면 뷰어 쪽 광고를 제거 할 수 있습니다. 

  - 8000 페이지를 소모 하시면 페이지 확보를 위한 30 초 광고와 설정 중간중간 에 있는 네이티브 광고를 제외한 모든 광고가 비활성화 됩니다. 

  - 8000 페이지 광고는 많은양의 변환 작업시 팝업되는 짧은 광고가 귀찮은 분을 위해 추가 하였습니다. 


13. 이어보기(히스토리 연동) 기능 설명 : 
  - 이어보기 기능을 지원 합니다. 
  
  - 예를 들면 A(1~50) 라는 제목의 문서가 있는데, 이번에 A(1~100) 이라는 문서를 새로 받았다고 가정 해 보겠습니다. 
  
  - 이경우 이미 본위치를 건너뛰어서 볼 필요가 있습니다. 
  
  - 내서재에서 텍스트 파일을 선택하면 이어보기 메뉴가 나옵니다.
  
  - 이 메뉴를 선택하신뒤에 히스토리에서 A(1~50)을 선택하면 자동으로 이전에 본 위치를 건너뛰게 됩니다. 

  - 나중에 시간되실때 히스토리에서 A(1~50) 을제거 하시기 바랍니다. 

  - 히스토리에서는 총 읽은 권수도 같이 집계가 됩니다. 
  
  - 같은 문서가 남아 있을경우 정확하지 않은 정보가 표시 될 수 있습니다. 



14. 북마크/메모/별점 설명 : 
  - 어플엔 메모와 북마크 /별점 기능을 지원 합니다. 

  - 북마크 설정에서 북마크 기능을 켜시면 텍스트를 길게 눌러 추가 할 수 있습니다. 

  - 메모 기능은 파일을 한번이라도 열어서 본경우에 파일을 선택하여 입력 할 수 있습니다. 

  - 별점은 파일을 한번이라도 열경우 파일 선택시 상단에 별 모양이 있습니다. 

  - 메모와 별점은 파일 자체에 대한 평가을 기록하는 용도로 사용 할 수 있습니다. 

  - 북마크는 중요내용을 기록하는 용도로 사용 할 수 있습니다. 



14. TTS 문장검사기 설명 : 
  - 수많은 문서를 보다보니 잘쓴글과 그렇지 않은 글의 패턴이 보이기 시작했습니다. 

  - 아쉬운 글에서는 항상 나오는 패턴을 발견하였습니다. 
  
  - 저는 이 패턴을 정리하여 각각 패턴별로 가중치를 두고 제작한 기능입니다. 

  - 처음엔 이기능을 [지뢰찾기]로 이름을 지었습니다.
  
  - 하지만, 이기능은 너무 주관적인 판단으로 만든 기능이라 TTS 문장 검사기로 변경하였습니다.

  - 이 기능은 한국어만 지원 합니다. 

  - 다른 국가의 언어를 제가 잘 모르기 때문에 좋은글인지 나쁜글인지 패턴을 정리 할 수가 없더군요. 

  - 혹시 타 국가에도 이런 패턴이 있다면 메일로 제보 부탁 드립니다. 
  
  - 여러 국가 언어를 지원할 수 있게되면 설정으로 기능을 이동 하겠습니다. 

  - 설정엔 아직 한자리 남아 있습니다. 



15. 개행정리 기능 설명 :
  - 개행정리는 문장이 끝나지 않았지만 엔터가 쳐져 있는 경우에 사용할 수 있는 방법 입니다. 

  - 보통 ocr 변환으로 작업된 문서가 이런 현상이 많이 발생 합니다. 

  - 이 기능은 단순히 엔터만 합쳐 주는 것이 아닙니다. 
  
  - 수많은 개행 오류가 있는 문서를 분석하고 그패턴을 정리하여 구현한 기능입니다. 

  - AI로 정리하는걸 도입 해려고 했지만 쉽지 않았네요. 
  
  - 요즘 자연어 처리 쪽에 연구가 많이 되고 있으니 추후 AI 로 처리 할 수 있으면 추가 해보도록 하겠습니다. 


16. 번역 기능 설명 : 
  - 로컬 번역 / 웹브라우져 번역 기능을 지원 합니다. 
  
  - 로컬 번역은 구글에서 개발한 AI 로 번역되는 기능입니다. 단순한 단어정도는 잘 번역 되지만 문장 번역에 오역이 상당히 많은것 같습니다. 

  - 웹브라우져 번역은 번역할 문장을 브라우져 로 띄워서 보여 주는 기능입니다. 

  - [설정 - 번역설정] 에서 번역 기능을 켜시고 문장을 길게 누를경우 번역기능이 작동 합니다. 


17. 기타 사항 : 
  - 이어플은 문서를 열때, 세번 이상 반복되는 엔터는 무조건 두번으로 바꿔서 열게 됩니다. 

  - 과도한 엔터는 TTS 로 듣기에도 불편하고 보기에도 불편 합니다. 

  - 제 어플을 장시간 쓰시면서 이미 눈치 채신분도 있을거고, "음? 그러고 보니 엔터 많이 들어간걸 본적이없네?" 라고 생각 하실 분도 있을것 같습니다. 

  - 텍스트 뷰어에서 지원할수 있는 기능을 최대한 구현 해 놓았습니다. 
  
  - TTS 를 위해 제작 하였고, TTS 를 위해 수많은 전처리 작업과 수많은 기능을 추가 하였습니다. 

  - 한국어 사용자의 경우 TTS 필터를 전부 다운로드 하시고 제목을 자세히 보시기 바랍니다. 
  
  - 원본 그대로 읽기보단 좀더 자연스러운 문장으로 바꿔서 읽도록 수많은 장치 를 만들어 놓을걸 볼 수 있습니다. 

  - 이 어플은 무료 입니다. 아직 유료 계획은 없습니다. 

  - 광고는 사용에 방해 되지 않도록만 넣었습니다. OCR 작업시 짧은 광고가 불편 할 수 있지만, 이런 광고는 제거 할 수 있도록 기능도 추가 해 놓았습니다. 

  - 이 어플을 처음 개발 하는데는 두달 정도 걸린것 같습니다. 처음엔 백그라운드 에서 TTS 가 안끊기는게 전부 였거든요. 
  
  - 그이후 개선 사항 받아서 개선하는데 2년 정도 걸린것 같네요. 

  - 아직도 개선 중이고, 신기술이 나오면 어플에 녹여 보려고 노력 하고 있습니다. 


열일곱 가지 사용법을 적었지만, 아직 절반도 못쓴것 같습니다. 

배경색, 배경이미지, 글자색, 페이지 넘김 방식 등 단순 기능은 생략 하였습니다. 

이 문서가 많은 분들에게 도움이 되시길 바랍니다. 

문의 사항은 다양한 채널을 통해 문의 하실 수 있습니다. 

[설정 - 정보 및 도움말] 에 보시면 카카오, 텔레그램 단톡방 링크가 있으니 궁금하신 점이 있다면 들어 오셔서 문의 하실 수 있습니다. 

가급적 익명으로 들어 오시기 바랍니다. 

끝까지 읽어 주셔서 감사합니다. 

이 어플이 삶에 도움이 되시길 바라며 이만 줄이겠습니다.

감사합니다.
