ວິທີການໃຊ້ມັນ:

ຄໍາຮ້ອງສະຫມັກນີ້ມີຫຼາຍຫນ້າທີ່, ສະນັ້ນຂ້າພະເຈົ້າຂໍແນະນໍາໃຫ້ທ່ານອ່ານມັນເທື່ອລະອັນ.

1. ວິທີການເບິ່ງເອກະສານ:
  - ກະລຸນາກົດທີ່ Add File ໃນເອກະສານຂອງຂ້ອຍເພື່ອເພີ່ມ txt.
  
  - ແຕະແຟ້ມ TXT ແລະກົດປຸ່ມເປີດຢູ່ດ້ານລຸ່ມເພື່ອເບິ່ງເອກະສານ.


2. ວິທີການຊ່ອນ Ads ຜູ້ຊົມ / ປຸ່ມນໍາທາງດ້ານລຸ່ມ:
  - ສອງເທົ່າຂອງສູນກາງຂອງຫນ້າຈໍ.
  
  - ຂະນະທີ່ທ່ານປ່ຽນໄປໃຊ້ຫນ້າຈໍເຕັມ, ໂຄສະນາແລະປຸ່ມຫາຍໄປ.


3. ເພື່ອປ່ຽນປຸ່ມ Mitt Mitt Stread
  - ທ່ານສາມາດປ່ຽນມັນພາຍໃຕ້ການຕັ້ງຄ່າ - ສໍາຜັດການຕັ້ງຄ່າພື້ນທີ່.
  
  - ທ່ານສາມາດກໍານົດຈໍານວນຫນ້າທີ່, ລວມທັງຫນ້າຈໍເຕັມ, ການເຄື່ອນໄຫວຂອງຫນ້າ, ການປັບຄວາມສະຫວ່າງ, ແລະຄວບຄຸມປະລິມານ, ແລະຄວບຄຸມປະລິມານ.
  
  - ການຄວບຄຸມໄລຍະຫ່າງໄກ Bluetooth ກໍ່ໄດ້ຮັບການສະຫນັບສະຫນູນ, ສະນັ້ນທ່ານສາມາດນອນຢູ່ແລະເບິ່ງມັນຢ່າງສະບາຍ.


4. ເພື່ອປ່ຽນຕົວອັກສອນຫຼືຕົວອັກສອນຂະຫນາດ:
  - ທ່ານສາມາດປ່ຽນຕົວອັກສອນຫລືຕົວອັກສອນຂະຫນາດພາຍໃຕ້ການຕັ້ງຄ່າ - ການຕັ້ງຄ່າ - ການຕັ້ງຄ່າຕົວອັກສອນ.
  
  - ທ່ານສາມາດໃຊ້ຕົວອັກສອນໂດຍກົດປຸ່ມດາວໂຫລດເພື່ອຮັບຕົວອັກສອນຟຣີຫຼາຍສິບຕົວອັກສອນ.


5. ວິທີການໃຊ້ການແປງຕົວຫນັງສືຮູບພາບ:
  - ທ່ານສາມາດສະກັດຂໍ້ຄວາມຈາກຮູບທີ່ບີບອັດໄດ້.
  
  - ນີ້ເອີ້ນວ່າຫນ້າທີ່ OCR.
  
  - ວາລະສານຂອງຂ້ອຍ - ກົດ Add File ແລະເພີ່ມແຟ້ມ zip.
  
  - ທ່ານສາມາດປ່ຽນໂດຍການເລືອກແຟ້ມ zip ແລະຫຼັງຈາກນັ້ນເລືອກເມນູ OR.
  
  - ຫນ້າເວັບແມ່ນບໍລິໂພກໃນລະຫວ່າງການປ່ຽນໃຈເຫລື້ອມໃສ.

  - OCR ເຮັດວຽກກັບ Ai, ສະນັ້ນຄວາມໄວແລະຄວາມຖືກຕ້ອງແມ່ນຂ້ອນຂ້າງດີ.


6. Epub, ວິທີການປ່ຽນໃຈເຫລື້ອມໃສ PDF:
  - ມັນກໍ່ຄືກັນກັບວິທີການປ່ຽນຮູບພາບ.

  - ມີບາງສິ່ງທີ່ຕ້ອງຮູ້.

  - EPUB ຫຼື PDF ອາດຈະເປັນ "ເຂົ້າລະຫັດ".

  - ມັນສ່ວນໃຫຍ່ແມ່ນໃຊ້ໃນການທໍາລາຍຕົວອັກສອນແລະເຮັດໃຫ້ພວກເຂົາເບິ່ງເຫັນໄດ້ໃນຕົວອັກສອນທີ່ແນ່ນອນເທົ່ານັ້ນ.

  - ໃນກໍລະນີນີ້, ມັນອອກມາຄືກັບຕົວລະຄອນນອກຕົວອັກສອນ, ບໍ່ແມ່ນພາສາເກົາຫຼີ, ສະນັ້ນທ່ານບໍ່ສາມາດອ່ານໄດ້ໃນ TTS.

  - ເປັນວິທີການແກ້ໄຂບັນຫານີ້, ຫນ້າທີ່ຂອງການສະກັດຮູບພາບສາມາດນໍາໃຊ້ໄດ້, ແລະຫຼັງຈາກນັ້ນ ORR ສາມາດຫມຸນໄດ້ເພື່ອສະກັດເອົາມັນ.


7. ວິທີການຕັ້ງຄ່າ TTS:
  - ການຕັ້ງຄ່າທີ່ກ່ຽວຂ້ອງກັບ TTS ແມ່ນສຸມໃສ່ໃນການຕັ້ງຄ່າ - ການຕັ້ງຄ່າ TTS, ການຕັ້ງຄ່າຕົວກອງ TNS.
  
  - ການຕັ້ງຄ່າ TTS ລວມມີຄຸນລັກສະນະ TTS ແບບງ່າຍດາຍ, ເຊັ່ນສຽງ, ຄວາມໂປ່ງໃສ, ຕົວອັກສອນສິ້ນສຸດລົງໃນແຕ່ລະຄັ້ງ, ພ້ອມທັງອຸປະກອນຕ່າງໆທີ່ສົມບູນກັບຄຸນລັກສະນະຂອງ TTS.
  
  - ຈໍານວນປະໂຫຍກອ່ານໃນເວລາທີ່ມີຫນ້າທີ່ເພີ່ມເຂົ້າໃນການອະນຸຍາດໃຫ້ມີຫລາຍສາຍທີ່ຈະອ່ານໃນເວລາດຽວກັນທີ່ຈະໄດ້ຮັບການຊັກຊ້າຫນ້ອຍທີ່ສຸດ.

  - ຕົວກອງ TNS ແມ່ນຫນຶ່ງໃນບັນດາຄຸນລັກສະນະທີ່ມີປະສິດທິພາບທີ່ສຸດຂອງໃບສະຫມັກນີ້.
  
  - [ການຕັ້ງຄ່າການກັ່ນຕອງ tts] ອະນຸຍາດໃຫ້ທ່ານສາມາດຕັ້ງຕົວອັກສອນພິເສດ, ຄໍາສັບຫລືປະໂຫຍກທີ່ທ່ານຕ້ອງການຂ້າມ.
  
  - ຖ້າທ່ານກົດປຸ່ມດາວໂຫລດ, ທ່ານສາມາດດາວໂຫລດລາຍການກັ່ນຕອງຫຼາຍສິບຕົວທີ່ຂ້ອຍຜະລິດໄດ້.
  
  - ທ່ານຍັງສາມາດເພີ່ມມັນເອງໄດ້ຖ້າທ່ານຕ້ອງການ.

  - [ຫນ້າທີ່ປິດແບບອັດຕະໂນມັດ] ສາມາດກໍານົດເວລາປິດອັດຕະໂນມັດໂດຍການສໍາຜັດກັບໄອຄອນທີ່ມີຮູບຊົງຂອງດວງຈັນຢູ່ເບື້ອງຊ້າຍຂອງປຸ່ມຢຸດເມື່ອຫຼີ້ນ TTS.


8. ລາຍລະອຽດຂອງຄວາມສາມາດສໍາຮອງແລະການຟື້ນຟູ:
  - ການຕັ້ງຄ່າ - ການສໍາຮອງແລະການກູ້ຄືນແມ່ນຄຸນນະສົມບັດທີ່ອະນຸຍາດໃຫ້ມີອຸປະກອນຫຼາຍຢ່າງໃນການນໍາໃຊ້ແອັບພລິເຄຊັນໃນສະພາບແວດລ້ອມດຽວກັນ.
  
  - ທ່ານສາມາດເຂົ້າສູ່ລະບົບໄດສໍາລັບການຟື້ນຟູສໍາຮອງ.
  
  - ເມື່ອເບິ່ງຈາກຫລາຍອຸປະກອນເຊັ່ນ: PC, ໂທລະສັບ, ໂທລະສັບ, ແລະ Pads, ທ່ານສາມາດໃຊ້ Backup ທີ່ຢູ່ເພື່ອ synchronize ສະຖານທີ່ຂອງທ່ານ.
  
  - ເລືອກອຸປະກອນທີ່ທ່ານຕ້ອງການທີ່ຈະອັດສະຈັນແລະເປີດຄຸນສົມບັດການອັດຕະໂນມັດເພື່ອຍ້າຍໄປທີ່ສະຖານທີ່ສຸດທ້າຍເມື່ອທ່ານສະຫມັກຫຼືກັບຄືນ.


9. ວິທີການໃຊ້ການດັດແປງຂໍ້ຄວາມ batch:
  - 1. [ກໍານົດຊຸດທີ່ກໍານົດໄວ້ - ສະແດງປຸ່ມດັດແກ້]
  
  - 2. [ຕັ້ງຄ່າ - ການດັດແປງຂໍ້ຄວາມສ່ວນໃຫຍ່]
  
  - ທ່ານສາມາດໃຊ້ການດັດແປງ batch ໃນສອງທາງ.
  
  - ໃນກໍລະນີຂອງເບີ 1, ມັນຖືກນໍາໃຊ້ເພື່ອຟັງ TTS ແລະເພື່ອລົບລ້າງຫຼືປ່ຽນປະໂຫຍກໃຫ້ກັບຕົວລະຄອນອື່ນຖ້າບາງປະໂຫຍກຫຼືຄໍາເວົ້າທີ່ຫນ້າລໍາຄານ.
  
  - ໃນກໍລະນີທີ່ມີເລກ 2, ທ່ານສາມາດສະກັດຂໍ້ຄວາມກັບ OCR ກັບ OCR ແລະເລືອກເອກະສານຂໍ້ຄວາມຈາກເອກະສານເພື່ອປະຕິບັດການແກ້ໄຂຮູບພາບ.
  
  - ທ່ານຍັງສາມາດໃຊ້ປຸ່ມດາວໂຫລດເພື່ອດາວໂຫລດແລະໃຊ້ຂໍ້ຜິດພາດໃນການຮັບຮູ້ທີ່ເກີດຂື້ນເລື້ອຍໆສໍາລັບການແກ້ໄຂຂໍ້ຄວາມຂອງ Batch.
  
  - ການແກ້ໄຂສ່ວນໃຫຍ່ - ການດາວໂຫລດຈະສະແດງບັນຊີລາຍຊື່ເທົ່ານັ້ນຖ້າການຕັ້ງຄ່າພາສາຢູ່ໃນພາສາເກົາຫຼີ.
  
  - ນີ້ແມ່ນຍ້ອນວ່ານັກພັດທະນາບໍ່ເວົ້າພາສາອື່ນ.
  
  - ຖ້າທ່ານລາຍງານການນໍາໃຊ້ຫນ້າທີ່ລາຍງານ, ຂ້ອຍຈະເພີ່ມພາສາຂອງປະເທດ.


10. ຖ້າມີປະໂຫຍກທີ່ງົມງາຍເມື່ອຟັງ TTS, ເຮັດວຽກອອກ:
  - ຖ້າມັນບໍ່ສໍາຄັນວ່າຕົ້ນສະບັບຖືກດັດແປງ, ກະລຸນາປ່ຽນມັນໃຫ້ເປັນ [ການດັດແກ້ທີ່ມີການປ່ຽນແປງ].

  - ຫຼັງຈາກການປ່ຽນແປງ, ເອກະສານສໍາຮອງຂໍ້ມູນຈະຍັງຄົງຢູ່ໃນຫໍສະມຸດຂອງຂ້ອຍ, ສະນັ້ນຖ້າທ່ານສາມາດດັດແປງມັນໂດຍບັງເອີນ, ທ່ານສາມາດກູ້ເອົາມັນອີກຄັ້ງ.

  - ຫນ້າຫນຶ່ງສາມາດຖືໄດ້ເຖິງ 60,000 ຫນ້າ.

  - ເມື່ອກໍາຈັດໂຄສະນາ, ທ່ານສາມາດໃຊ້ຫນ້າເພື່ອກໍາຈັດການໂຄສະນາໃນໄລຍະເວລາໃດຫນຶ່ງ. (ຍົກເວັ້ນເວລານອນ, ມັນປະມານຫນຶ່ງມື້.)

  - ໃນເວລາທີ່ທ່ານບໍລິໂພກ 8000 ຫນ້າ, ໂຄສະນາທັງຫມົດຖືກປິດໃຊ້ງານຍົກເວັ້ນການໂຄສະນາ 30 ວິນາທີສໍາລັບການຊື້ຂອງຫນ້າແລະການໂຄສະນາພື້ນເມືອງຢູ່ເຄິ່ງກາງຂອງການຕັ້ງຄ່າ.

  - 8000 ໂຄສະນາ ADS ໄດ້ຖືກເພີ່ມເຂົ້າສໍາລັບຜູ້ທີ່ກັງວົນໂດຍການໂຄສະນາສັ້ນທີ່ປະກົດຕົວໃນລະຫວ່າງການປ່ຽນໃຈເຫລື້ອມໃສເປັນຈໍານວນຫລວງຫລາຍ.


13. ລາຍລະອຽດຄຸນນະສົມບັດສໍາລັບຫູ (ປະຫວັດສາດໃນລະດູການເວລາ):
  - ມັນສະຫນັບສະຫນູນຫນ້າທີ່ເບິ່ງຫູ.
  
  ຍົກຕົວຢ່າງ, ໃຫ້ເວົ້າວ່າພວກເຮົາມີເອກະສານທີ່ມີຊື່ວ່າ A (1 ເຖິງ 50), ແລະພວກເຮົາມີເອກະສານໃຫມ່ທີ່ເອີ້ນວ່າ A (1 ເຖິງ 100).
  
  - ໃນກໍລະນີນີ້, ມັນມີຄວາມຈໍາເປັນແລ້ວທີ່ຈະຂ້າມຕໍາແຫນ່ງຫຼັກ.
  
  - ເມື່ອທ່ານເລືອກແຟ້ມຂໍ້ຄວາມໃນຫ້ອງສະມຸດຂອງຂ້ອຍ, ເມນູ View View ຈະປາກົດຂຶ້ນ.
  
  - ຖ້າທ່ານເລືອກເມນູນີ້ແລະຫຼັງຈາກນັ້ນເລືອກ A (1 ເຖິງ 50) ໃນປະຫວັດສາດ, ມັນຈະຂ້າມຕໍາແຫນ່ງທີ່ເບິ່ງກ່ອນຫນ້ານີ້ໂດຍອັດຕະໂນມັດ.

  - ເມື່ອທ່ານມີເວລາຕໍ່ມາ, ກະລຸນາເອົາ A (1-50) ຈາກປະຫວັດສາດ.

  - ໃນປະຫວັດສາດ, ຈໍານວນປື້ມທັງຫມົດທີ່ອ່ານກໍ່ຖືກນັບ.
  
  - ຖ້າເອກະສານດຽວກັນຍັງຄົງຢູ່, ຂໍ້ມູນທີ່ບໍ່ຖືກຕ້ອງອາດຈະຖືກສະແດງ.

14. ອະທິບາຍກ່ຽວກັບປະໂຫຍກ TTS:

  - ຂ້າພະເຈົ້າໄດ້ຈັດຮູບແບບນີ້ແລະຜະລິດມັນໂດຍການຊັ່ງນໍ້າໃນແຕ່ລະຮູບແບບ.

  - ພວກເຮົາໄດ້ຕັ້ງຊື່ມັນ [ຄົ້ນຫາບໍ່ແຮ່] ໃນຕອນທໍາອິດ.

  - ຂ້ອຍບໍ່ຮູ້ພາສາຂອງປະເທດອື່ນ, ສະນັ້ນຂ້ອຍບໍ່ສາມາດຈັດຮູບແບບຂອງຂ້ອຍໄດ້ບໍ່ວ່າມັນຈະດີຫຼືບໍ່ດີ.

  - ເມື່ອທ່ານສາມາດສະຫນັບສະຫນູນຫລາຍພາສາຂອງຊາດ, ພວກເຮົາຈະຍ້າຍຫນ້າທີ່ໄປຫາການຕັ້ງຄ່າ.

  - ຍັງມີຈຸດຫນຶ່ງທີ່ຍັງເຫຼືອຢູ່ໃນການຕັ້ງຄ່າ.
